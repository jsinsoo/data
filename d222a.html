2. 수업을 여는 멘트

여름이 되면 아이스크림 판매량이 늘어납니다. 또 공부 시간이 늘어나면 시험 점수가 오를 가능성도 있습니다. 이처럼 한 변수가 커질 때 다른 변수도 함께 커지거나 작아지는 경향을 우리는 상관 관계라고 부릅니다. 하지만 “기온이 오르면 아이스크림 판매량이 오른다”는 말과 “아이스크림을 많이 팔았기 때문에 기온이 올랐다”는 말은 전혀 다릅니다. 오늘은 데이터를 보고 두 변수의 관계를 찾되, 성급하게 원인과 결과로 단정하지 않는 방법을 배웁니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 상관 관계

상관 관계는 두 변수가 함께 변하는 경향을 말합니다. 한 변수가 증가할 때 다른 변수도 증가하거나 감소하는 경향이 있는지 살펴봅니다.

개념 2. 양의 상관

양의 상관은 한 변수가 증가할 때 다른 변수도 증가하는 경향입니다. 예를 들어 기온이 높아질수록 아이스크림 판매량이 증가할 수 있습니다.

개념 3. 음의 상관

음의 상관은 한 변수가 증가할 때 다른 변수는 감소하는 경향입니다. 예를 들어 운동 시간이 늘수록 특정 조건에서 체지방률이 감소하는 경향이 나타날 수 있습니다.

개념 4. 무상관

무상관은 두 변수 사이에 뚜렷한 함께 변하는 경향이 보이지 않는 경우입니다. 산점도에서 점들이 특별한 방향 없이 흩어져 보입니다.

개념 5. 산점도

산점도는 두 변수의 값을 좌표평면 위의 점으로 나타낸 그래프입니다. x축에는 한 변수, y축에는 다른 변수를 놓고 점들의 방향과 밀집 정도를 관찰합니다.

개념 6. 상관계수

상관계수는 두 변수의 선형 관계의 방향과 강도를 수치로 나타낸 값입니다. 보통 -1에서 1 사이의 값을 가지며, 1에 가까우면 강한 양의 상관, -1에 가까우면 강한 음의 상관, 0에 가까우면 선형 관계가 약하다고 해석합니다.

\[ -1 \le r \le 1 \]

개념 7. 피어슨 상관계수

피어슨 상관계수는 두 수치형 변수 사이의 선형 관계를 측정하는 대표적인 방법입니다. 직선적인 관계에는 유용하지만 곡선 형태의 관계를 항상 잘 포착하지는 못합니다.

개념 8. 이상치의 영향

이상치는 상관계수와 산점도 해석에 큰 영향을 줄 수 있습니다. 한두 개의 극단적인 점 때문에 상관이 강해 보이거나 약해 보일 수 있습니다.

개념 9. 상관은 인과가 아니다

상관 관계가 있다는 것은 두 변수가 함께 변하는 경향이 있다는 뜻이지, 한 변수가 다른 변수의 원인이라는 뜻은 아닙니다. 숨은 변수나 우연한 관계가 있을 수 있습니다.

개념 10. 예측과 상관 관계

상관 관계는 데이터 기반 예측의 출발점입니다. 두 변수 사이에 강한 관련성이 있으면 하나의 변수로 다른 변수의 경향을 예측하는 모델을 만들 수 있습니다. 그러나 예측은 항상 불확실성을 포함합니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 양의 상관 데이터 확인

temperature = [20, 22, 25, 27, 30, 32]
icecream = [30, 35, 42, 50, 65, 72]

for x, y in zip(temperature, icecream):
    print("기온:", x, "판매량:", y)

설명: 기온이 높아질수록 아이스크림 판매량도 증가하는 경향이 있습니다.

코드 2. 산점도 그리기

import matplotlib.pyplot as plt

temperature = [20, 22, 25, 27, 30, 32]
icecream = [30, 35, 42, 50, 65, 72]

plt.scatter(temperature, icecream)
plt.title("기온과 아이스크림 판매량")
plt.xlabel("기온")
plt.ylabel("판매량")
plt.show()

설명: 산점도는 두 변수의 관계를 눈으로 확인하는 기본 그래프입니다.

코드 3. 상관계수 직접 계산하기

x = [1, 2, 3, 4, 5]
y = [2, 4, 5, 8, 10]

mean_x = sum(x) / len(x)
mean_y = sum(y) / len(y)

num = sum((a - mean_x) * (b - mean_y) for a, b in zip(x, y))
den_x = sum((a - mean_x) ** 2 for a in x) ** 0.5
den_y = sum((b - mean_y) ** 2 for b in y) ** 0.5

r = num / (den_x * den_y)
print("상관계수:", r)

설명: 두 변수의 편차가 함께 움직이는 정도를 이용해 상관계수를 계산합니다.

코드 4. numpy로 상관계수 구하기

import numpy as np

study_hours = [1, 2, 3, 4, 5, 6]
scores = [55, 62, 70, 78, 86, 92]

corr = np.corrcoef(study_hours, scores)[0, 1]

print("공부 시간과 점수의 상관계수:", corr)

설명: numpy의 corrcoef를 사용하면 상관계수를 간단히 구할 수 있습니다.

코드 5. 음의 상관 산점도

import matplotlib.pyplot as plt

exercise = [1, 2, 3, 4, 5, 6]
fat_rate = [30, 28, 25, 23, 21, 19]

plt.scatter(exercise, fat_rate)
plt.title("운동 시간과 체지방률")
plt.xlabel("운동 시간")
plt.ylabel("체지방률")
plt.show()

설명: 운동 시간이 늘어날수록 체지방률이 감소하는 경향은 음의 상관으로 볼 수 있습니다.

코드 6. 이상치가 상관계수에 미치는 영향

import numpy as np

x1 = [1, 2, 3, 4, 5]
y1 = [2, 4, 6, 8, 10]

x2 = [1, 2, 3, 4, 5, 20]
y2 = [2, 4, 6, 8, 10, 1]

print("이상치 없음:", np.corrcoef(x1, y1)[0, 1])
print("이상치 있음:", np.corrcoef(x2, y2)[0, 1])

설명: 하나의 극단적인 값이 상관계수를 크게 바꿀 수 있습니다.

코드 7. 상관과 인과 주의 메시지

corr = 0.92

if abs(corr) >= 0.8:
    print("강한 상관 관계가 있습니다.")
    print("하지만 이것만으로 인과 관계를 단정할 수는 없습니다.")
else:
    print("상관 관계가 약하거나 뚜렷하지 않습니다.")

설명: 상관계수가 높아도 원인과 결과 관계를 증명하려면 추가적인 근거가 필요합니다.

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 공부 시간과 시험 점수

import numpy as np

study = [1, 2, 3, 4, 5, 6]
score = [55, 62, 70, 78, 86, 92]

r = np.corrcoef(study, score)[0, 1]
print("상관계수:", r)

교육 데이터공부 시간과 점수 사이에 양의 상관이 있는지 확인하여 학습 습관 분석에 활용할 수 있습니다.

예제 2. 기온과 아이스크림 판매량 산점도

import matplotlib.pyplot as plt

temp = [20, 22, 25, 27, 30, 32]

plt.ylabel("판매량")

판매 데이터기온이 오를수록 판매량이 증가하는 경향이 있는지 시각적으로 확인할 수 있습니다.

예제 3. 운동 시간과 체지방률

import numpy as np

exercise = [1, 2, 3, 4, 5, 6]
fat = [30, 28, 25, 23, 21, 19]

r = np.corrcoef(exercise, fat)[0, 1]

건강 데이터운동 시간이 늘수록 체지방률이 감소하는 경향이 있다면 음의 상관을 생각할 수 있습니다.

예제 4. 수면 시간과 집중도

import matplotlib.pyplot as plt
import numpy as np

sleep = [4, 5, 6, 7, 8, 9]
focus = [50, 60, 72, 80, 83, 84]

print("상관계수:", np.corrcoef(sleep, focus)[0, 1])

생활 데이터수면 시간이 늘어날수록 집중도가 높아지는 경향이 있는지 탐색할 수 있습니다.

예제 5. 이상치 포함 전후 비교

import numpy as np

x_normal = [1, 2, 3, 4, 5]
y_normal = [2, 4, 6, 8, 10]

x_outlier = [1, 2, 3, 4, 5, 10]
y_outlier = [2, 4, 6, 8, 10, 1]

print("이상치 없음:", np.corrcoef(x_normal, y_normal)[0, 1])
print("이상치 있음:", np.corrcoef(x_outlier, y_outlier)[0, 1])

데이터 품질상관 분석 전에 이상치를 확인해야 관계 해석이 왜곡되는 것을 줄일 수 있습니다.

Closing Talk

8. 수업을 마무리하는 멘트

오늘 우리는 두 변수 사이의 관계를 상관 관계라는 개념으로 살펴보았습니다. 산점도는 관계를 눈으로 확인하게 해 주고, 상관계수는 관계의 방향과 강도를 수치로 표현합니다. 그러나 상관 관계는 인과 관계를 자동으로 증명하지 않습니다. 컴퓨터과학은 파이썬으로 산점도와 상관계수를 빠르게 계산하게 해 주고, 수학은 좌표평면과 상관계수의 원리를 설명합니다. 과학은 변수 사이 관계를 가설로 세우고 실험과 검증을 통해 원인과 결과를 조심스럽게 판단하게 합니다. 데이터 과학과 머신러닝에서 상관 관계는 예측에 사용할 중요한 변수 관계를 찾는 출발점입니다.

컴퓨터과학과의 연결

파이썬, numpy, matplotlib을 활용해 변수 관계를 계산하고 시각화합니다.

수학과의 연결

좌표평면, 편차, 공분산, 상관계수는 두 변수의 관계를 수치로 표현하는 수학적 도구입니다.

과학과의 연결

상관 관계는 가설을 세우는 단서가 되지만, 인과 관계는 실험과 통제를 통해 검증해야 합니다.

머신러닝과의 연결

상관 관계 분석은 예측 모델에서 사용할 입력 변수를 선택하고 데이터 패턴을 이해하는 데 도움을 줍니다.
