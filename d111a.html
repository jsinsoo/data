Opening Talk

2. 수업을 여는 멘트

여러분은 오늘 아침에 휴대폰을 몇 번 확인했나요? 날씨 앱은 비가 올지 알려 주고, 영상 플랫폼은 내가 좋아할 만한 영상을 추천해 줍니다. 쇼핑몰은 내가 관심 가질 상품을 보여 주고, 지도 앱은 가장 빠른 길을 찾아 줍니다. 이 모든 과정의 공통점은 바로 데이터입니다. 오늘 수업에서는 단순한 숫자와 기록이 어떻게 의미 있는 정보가 되고, 또 그것이 의사결정과 진로 선택까지 연결되는지 배워 보겠습니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 데이터(Data)

데이터는 관찰, 측정, 기록을 통해 얻은 값입니다. 예를 들어 학생의 키, 시험 점수, 하루 걸음 수, 기온, 미세먼지 농도, 검색 기록, 사진, 음성 등이 모두 데이터가 될 수 있습니다. 데이터는 그 자체로는 단순한 값이지만, 분석 과정을 거치면 의미 있는 정보가 됩니다.

개념 2. 정보(Information)

정보는 데이터를 목적에 맞게 정리하고 해석하여 의미를 부여한 결과입니다. 예를 들어 “기온 31도”는 데이터이고, “오늘은 매우 더우므로 야외 활동 시 물을 충분히 마셔야 한다”는 정보입니다.

개념 3. 데이터 과학(Data Science)

데이터 과학은 데이터를 수집, 정제, 분석, 시각화하고 이를 바탕으로 문제를 해결하거나 예측하는 학문입니다. 컴퓨터과학의 프로그래밍, 수학의 통계와 확률, 과학의 탐구 방법이 함께 사용됩니다.

데이터 과학의 일반적인 흐름은 다음과 같습니다.

\[ \text{문제 정의} \rightarrow \text{데이터 수집} \rightarrow \text{데이터 정제} \rightarrow \text{분석} \rightarrow \text{시각화} \rightarrow \text{의사결정} \]

개념 4. 데이터 분석

데이터 분석은 데이터 속에서 패턴, 관계, 차이, 변화 경향을 찾아내는 과정입니다. 예를 들어 월별 전기 사용량을 분석하면 어느 계절에 전력 사용이 증가하는지 알 수 있습니다. 이를 통해 에너지 절약 계획을 세울 수 있습니다.

개념 5. 데이터 시각화

데이터 시각화는 숫자로 된 데이터를 그래프나 차트로 표현하여 한눈에 이해하기 쉽게 만드는 방법입니다. 막대그래프는 항목 간 비교에, 선그래프는 시간에 따른 변화에, 산점도는 두 변수 사이의 관계를 볼 때 유용합니다.

개념 6. 데이터 기반 의사결정

데이터 기반 의사결정은 감이나 추측이 아니라 실제 자료를 근거로 판단하는 방식입니다. 예를 들어 학교 매점에서 가장 많이 팔리는 음료 데이터를 분석하면 재고를 더 효율적으로 준비할 수 있습니다.

개념 7. 데이터 과학과 진로

데이터 과학은 다양한 직업과 연결됩니다. 데이터 분석가, 인공지능 개발자, 생명정보학자, 금융 분석가, 마케팅 분석가, 스포츠 데이터 분석가, 기후 데이터 연구자 등이 대표적입니다. 중요한 점은 데이터 과학이 특정 직업 하나가 아니라 거의 모든 분야에서 필요한 문제 해결 도구라는 것입니다.

Python Codes

4. 각각의 개념과 관련된 파이썬 코드

코드 1. 데이터와 정보 구분하기

# 데이터: 단순히 기록된 값
temperature = 31

# 정보: 데이터를 해석하여 의미를 부여한 결과
if temperature >= 30:
    print("오늘은 덥습니다. 물을 충분히 마시세요.")
else:
    print("활동하기 비교적 좋은 날씨입니다.")

설명: 31이라는 숫자는 데이터이고, 조건문을 통해 “덥다”라는 의미를 부여하면 정보가 됩니다.

코드 2. 평균을 이용한 데이터 분석

scores = [82, 91, 75, 88, 96]

average = sum(scores) / len(scores)

print("점수 목록:", scores)
print("평균 점수:", average)

설명: 여러 점수 데이터를 평균으로 요약하면 전체적인 학습 수준을 파악할 수 있습니다.

코드 3. 최댓값과 최솟값 찾기

temperatures = [24, 27, 30, 29, 26, 31, 28]

print("가장 높은 기온:", max(temperatures))
print("가장 낮은 기온:", min(temperatures))

설명: 최댓값과 최솟값은 데이터의 범위를 이해하는 데 도움을 줍니다.

코드 4. 간단한 데이터 시각화

import matplotlib.pyplot as plt

days = ["월", "화", "수", "목", "금"]
steps = [5200, 6100, 7300, 6900, 8100]

plt.bar(days, steps)
plt.title("요일별 걸음 수")
plt.xlabel("요일")
plt.ylabel("걸음 수")
plt.show()

설명: 막대그래프를 사용하면 요일별 걸음 수 차이를 쉽게 비교할 수 있습니다.

코드 5. 진로 추천을 위한 간단한 규칙 기반 프로그램

interest = input("관심 분야를 입력하세요(수학/코딩/생명/경제/스포츠): ")

if interest == "수학":
    print("추천 진로: 데이터 분석가, 통계학자")
elif interest == "코딩":
    print("추천 진로: 인공지능 개발자, 데이터 엔지니어")
elif interest == "생명":
    print("추천 진로: 생명정보학자, 의료 데이터 분석가")
elif interest == "경제":
    print("추천 진로: 금융 데이터 분석가, 마케팅 분석가")
elif interest == "스포츠":
    print("추천 진로: 스포츠 데이터 분석가")
else:
    print("다양한 분야에서 데이터 과학을 활용할 수 있습니다.")

설명: 실제 인공지능은 더 복잡한 데이터를 사용하지만, 이 예제는 입력값에 따라 진로를 추천하는 기본 원리를 보여 줍니다.

5. 진단평가 OX 예제 10개 + 상세풀이

1. 데이터는 관찰하거나 측정하여 얻은 값이다. (O/X)

정답: O
데이터는 기온, 점수, 키, 클릭 수처럼 관찰·측정·기록된 값을 의미합니다.

2. 정보는 아무 해석이 없는 숫자 그 자체만을 뜻한다. (O/X)

정답: X
정보는 데이터를 목적에 맞게 정리하고 해석하여 의미를 부여한 결과입니다.

3. 데이터 과학은 컴퓨터과학, 수학, 과학적 탐구 방법과 관련이 있다. (O/X)

정답: O
데이터 과학은 프로그래밍, 통계, 확률, 실험과 관찰을 함께 활용합니다.

4. 데이터 분석은 데이터 속에서 패턴과 관계를 찾는 과정이다. (O/X)

정답: O
예를 들어 기온과 아이스크림 판매량의 관계를 찾는 것이 데이터 분석입니다.

5. 데이터 시각화는 데이터를 더 어렵게 만들기 위한 과정이다. (O/X)

정답: X
시각화는 그래프와 차트를 이용해 데이터를 더 쉽게 이해하도록 돕습니다.

6. 데이터 기반 의사결정은 감이나 추측보다 실제 자료를 중시한다. (O/X)

정답: O
데이터 기반 의사결정은 근거 있는 판단을 가능하게 합니다.

7. 데이터 과학은 오직 컴퓨터 관련 직업에서만 사용된다. (O/X)

정답: X
의료, 금융, 교육, 스포츠, 환경, 예술 등 다양한 분야에서 활용됩니다.

8. 평균은 여러 데이터를 하나의 대표값으로 요약하는 방법 중 하나이다. (O/X)

정답: O
평균은 전체 데이터를 대표하는 값으로 자주 사용됩니다.

9. 데이터가 많으면 무조건 올바른 결론을 얻을 수 있다. (O/X)

정답: X
데이터가 많아도 편향, 오류, 잘못된 해석이 있으면 잘못된 결론이 나올 수 있습니다.

10. 데이터 과학자는 문제를 정의하고 데이터를 분석하여 해결 방안을 찾는다. (O/X)

정답: O
데이터 과학자는 단순히 코딩만 하는 것이 아니라 문제 해결 전체 과정을 다룹니다.

① 느낌 ② 기온 30도 ③ 행복 ④ 상상 ⑤ 의견만 있는 주장

정답: ②
기온 30도는 측정된 값이므로 데이터입니다.

2. 데이터 과학의 일반적인 과정으로 가장 적절한 것은?

① 결론 → 수집 → 문제 정의 ② 문제 정의 → 수집 → 분석 → 의사결정 ③ 시각화 → 삭제 → 추측 ④ 코딩 → 암기 → 종료 ⑤ 감정 → 판단 → 기록

정답: ②
데이터 과학은 문제를 정하고, 데이터를 수집·정제·분석한 뒤 의사결정으로 연결합니다.

3. 데이터 시각화에 대한 설명으로 옳은 것은?

① 데이터를 숨긴다 ② 데이터를 삭제한다 ③ 데이터를 그래프 등으로 표현한다 ④ 데이터를 무작위로 바꾼다 ⑤ 데이터를 모두 문자로만 바꾼다

정답: ③
시각화는 그래프, 차트 등을 사용해 데이터를 이해하기 쉽게 표현하는 과정입니다.

4. 시간에 따른 기온 변화를 나타내기에 가장 적절한 그래프는?

① 원그래프 ② 선그래프 ③ 표지만 ④ 그림일기 ⑤ 순서 없는 목록

정답: ②
시간에 따른 변화는 선그래프로 표현하면 흐름을 파악하기 쉽습니다.

5. 데이터 기반 의사결정의 장점으로 가장 적절한 것은?

① 근거 없는 판단 증가 ② 감정만 사용 ③ 실제 자료를 근거로 판단 ④ 결과를 숨김 ⑤ 분석을 피함

정답: ③
데이터 기반 의사결정은 실제 자료에 근거하므로 더 객관적인 판단을 도와줍니다.

6. 다음 중 데이터 과학과 관련된 진로가 아닌 것은?

① 데이터 분석가 ② 인공지능 개발자 ③ 생명정보학자 ④ 스포츠 데이터 분석가 ⑤ 데이터와 전혀 관련 없는 무작위 선택가

정답: ⑤
①~④는 모두 데이터 과학과 관련된 대표적인 진로입니다.

7. 파이썬 코드 sum(scores) / len(scores)가 구하는 값은?

① 최댓값 ② 최솟값 ③ 평균 ④ 문자열 ⑤ 그래프 제목

정답: ③
sum()은 합계, len()은 개수이므로 합계를 개수로 나누면 평균입니다.

8. 데이터 정제가 필요한 이유로 가장 적절한 것은?

① 오류와 결측값을 줄이기 위해 ② 데이터를 무조건 삭제하기 위해 ③ 그래프를 없애기 위해 ④ 컴퓨터를 끄기 위해 ⑤ 문제를 만들지 않기 위해

정답: ①
데이터에 오류나 빈 값이 있으면 분석 결과가 왜곡될 수 있으므로 정제가 필요합니다.

9. 데이터 과학을 배울 때 수학이 필요한 이유로 가장 적절한 것은?

① 색칠만 하기 위해 ② 통계와 확률을 이해하기 위해 ③ 키보드를 고치기 위해 ④ 인터넷 속도를 높이기 위해 ⑤ 파일 이름을 꾸미기 위해

정답: ②
평균, 분산, 확률, 상관관계 등은 데이터 분석의 중요한 수학적 기초입니다.

Real Life Coding

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 일주일 걸음 수 분석

steps = [5200, 6100, 7300, 6900, 8100, 9000, 7600]

average = sum(steps) / len(steps)
best = max(steps)

print("평균 걸음 수:", average)
print("가장 많이 걸은 날의 걸음 수:", best)

건강 데이터 스마트워치나 휴대폰에서 수집한 걸음 수 데이터를 이용하면 자신의 활동량을 파악할 수 있습니다. 평균과 최댓값을 통해 생활 습관을 점검할 수 있습니다.

예제 2. 하루 공부 시간 분석

study_hours = [2, 3, 1, 4, 2, 5, 3]

total = sum(study_hours)
average = total / len(study_hours)

print("일주일 총 공부 시간:", total)
print("하루 평균 공부 시간:", average)

학습 데이터 공부 시간 데이터를 기록하면 자신의 학습 패턴을 파악할 수 있습니다. 단순한 감이 아니라 실제 기록을 바탕으로 학습 계획을 조정할 수 있습니다.

예제 3. 매점 인기 상품 찾기

sales = {
    "물": 35,
    "주스": 28,
    "빵": 42,
    "우유": 19,
    "과자": 31
}

popular = max(sales, key=sales.get)

print("가장 많이 팔린 상품:", popular)
print("판매 개수:", sales[popular])

경영 데이터 상품별 판매량을 분석하면 어떤 상품을 더 많이 준비해야 하는지 판단할 수 있습니다. 이것이 데이터 기반 의사결정의 좋은 예입니다.

예제 4. 미세먼지 상태 판단

dust = 82

if dust <= 30:
    print("미세먼지 좋음")
elif dust <= 80:
    print("미세먼지 보통")
elif dust <= 150:
    print("미세먼지 나쁨")
else:
    print("미세먼지 매우 나쁨")

환경 데이터 미세먼지 수치를 기준에 따라 해석하면 외출 여부나 마스크 착용 여부를 결정할 수 있습니다. 숫자 데이터가 생활 정보로 바뀌는 과정입니다.

예제 5. 관심 분야에 따른 진로 추천

interests = ["코딩", "수학", "생명"]

if "코딩" in interests and "수학" in interests:
    print("추천 진로: 인공지능 개발자 또는 데이터 과학자")
elif "생명" in interests:
    print("추천 진로: 생명정보학자")
else:
    print("추천 진로: 데이터 활용 분야를 더 탐색해 보세요.")

진로 데이터 학생의 관심 분야를 데이터로 보고 조건에 따라 진로를 추천할 수 있습니다. 실제 진로 추천 시스템은 더 많은 데이터와 알고리즘을 사용합니다.

8. 수업을 마무리하는 멘트

오늘 우리는 데이터가 단순한 숫자가 아니라 세상을 이해하는 도구라는 것을 배웠습니다. 컴퓨터과학은 데이터를 처리하는 방법을 제공하고, 수학은 데이터를 분석하는 논리를 제공하며, 과학은 데이터를 통해 가설을 검증하는 태도를 제공합니다. 데이터 과학은 이 세 영역이 만나는 지점에 있습니다. 앞으로 여러분이 의사가 되든, 과학자가 되든, 교사가 되든, 개발자가 되든, 예술가가 되든 데이터는 문제를 발견하고 해결하는 중요한 언어가 될 것입니다.
