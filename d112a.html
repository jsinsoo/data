2. 수업을 여는 멘트

여러분이 도서관에서 책을 검색하거나, 쇼핑몰에서 상품을 찾거나, 학교 급식표를 확인할 때 사실은 데이터베이스를 사용하고 있습니다. 수많은 책, 상품, 학생, 성적, 출석 정보가 아무렇게나 저장되어 있다면 원하는 정보를 찾기 어렵습니다. 그래서 우리는 데이터를 종류별로 분류하고, 테이블에 정리하고, SQL이라는 언어로 필요한 정보만 정확히 찾아냅니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 데이터

데이터는 관찰, 측정, 기록을 통해 얻은 원자료입니다. 학생 이름, 출석 날짜, 시험 점수, 상품 가격, 기온, 사진, 댓글 등이 데이터입니다.

개념 2. 정보와 지식

정보는 데이터를 목적에 맞게 정리하고 해석한 결과입니다. 지식은 여러 정보를 연결하여 판단이나 예측에 활용할 수 있는 이해입니다.

\[ \text{데이터} \rightarrow \text{정보} \rightarrow \text{지식} \rightarrow \text{의사결정} \]

개념 3. 정형 데이터와 비정형 데이터

정형 데이터는 행과 열이 있는 표 형태의 데이터입니다. 비정형 데이터는 사진, 음성, 동영상, 자유 텍스트처럼 표 형태로 바로 정리하기 어려운 데이터입니다.

개념 4. 수치형 데이터와 범주형 데이터

수치형 데이터는 계산 가능한 숫자 데이터이고, 범주형 데이터는 종류나 분류를 나타내는 데이터입니다.

개념 5. 데이터베이스

데이터베이스는 관련 있는 데이터를 체계적으로 저장하고 관리하는 시스템입니다. 학교 성적 관리, 도서관 도서 검색, 병원 진료 기록 등이 데이터베이스로 관리됩니다.

개념 6. 테이블, 레코드, 필드

테이블은 데이터를 행과 열로 저장한 구조입니다. 레코드는 한 행의 데이터이고, 필드는 한 열의 속성입니다.

개념 7. 기본키

기본키는 각 레코드를 구별하는 고유한 값입니다. 학생번호, 도서번호, 주문번호처럼 중복되지 않는 값이 기본키가 될 수 있습니다.

개념 8. SQL

SQL은 데이터베이스에 저장된 데이터를 조회, 삽입, 수정, 삭제하기 위한 언어입니다. 가장 기본적인 조회문은 SELECT입니다.

개념 9. SELECT와 WHERE

SELECT는 어떤 열을 볼지 정하고, WHERE는 어떤 조건의 행을 가져올지 정합니다.

개념 10. ORDER BY와 GROUP BY

ORDER BY는 데이터를 정렬할 때 사용하고, GROUP BY는 같은 종류끼리 묶어 집계할 때 사용합니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 데이터와 정보 구분하기

score = 92

if score >= 90:
    info = "우수한 성취 수준"
else:
    info = "추가 학습 필요"

print("데이터:", score)
print("정보:", info)

설명: 92라는 점수는 데이터이고, 기준에 따라 해석한 “우수한 성취 수준”은 정보입니다.

코드 2. 데이터 유형 분류하기

data = {
    "이름": "민수",
    "점수": 88,
    "교통수단": "버스",
    "키": 171.5
}

for key, value in data.items():
    print(key, value, type(value))

설명: 문자열과 숫자 자료형을 확인하며 범주형 데이터와 수치형 데이터를 구분할 수 있습니다.

코드 3. SQLite 데이터베이스 만들기

import sqlite3

conn = sqlite3.connect("school.db")
cur = conn.cursor()

cur.execute("CREATE TABLE IF NOT EXISTS students (id INTEGER PRIMARY KEY, name TEXT, grade INTEGER, score INTEGER)")

conn.commit()
conn.close()

설명: SQLite를 사용해 학생 데이터를 저장할 테이블을 만듭니다. id는 기본키입니다.

코드 4. 데이터 삽입하기

import sqlite3

conn = sqlite3.connect("school.db")
cur = conn.cursor()

cur.execute("INSERT INTO students VALUES (1, '민수', 1, 88)")
cur.execute("INSERT INTO students VALUES (2, '지우', 1, 95)")
cur.execute("INSERT INTO students VALUES (3, '하린', 2, 76)")

conn.commit()
conn.close()

설명: INSERT INTO는 테이블에 새로운 행을 추가합니다.

코드 5. SELECT로 전체 조회하기

import sqlite3

conn = sqlite3.connect("school.db")
cur = conn.cursor()

cur.execute("SELECT * FROM students")
rows = cur.fetchall()

for row in rows:
    print(row)

conn.close()

설명: SELECT * FROM students는 students 테이블의 모든 열과 행을 조회합니다.

코드 6. WHERE 조건으로 조회하기

import sqlite3

conn = sqlite3.connect("school.db")
cur = conn.cursor()

cur.execute("SELECT name, score FROM students WHERE score >= 90")
rows = cur.fetchall()

for row in rows:
    print(row)

conn.close()

설명: WHERE score >= 90은 점수가 90점 이상인 학생만 조회합니다.

코드 7. ORDER BY로 정렬하기

import sqlite3

conn = sqlite3.connect("school.db")
cur = conn.cursor()

cur.execute("SELECT name, score FROM students ORDER BY score DESC")
rows = cur.fetchall()

for row in rows:
    print(row)

conn.close()

설명: ORDER BY score DESC는 점수를 높은 순서로 정렬합니다.

코드 8. GROUP BY로 학년별 평균 구하기

import sqlite3

conn = sqlite3.connect("school.db")
cur = conn.cursor()

cur.execute("SELECT grade, AVG(score) FROM students GROUP BY grade")
rows = cur.fetchall()

for row in rows:
    print(row)

conn.close()

설명: GROUP BY grade는 학년별로 데이터를 묶고, AVG(score)로 평균 점수를 계산합니다.

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 도서관 도서 데이터베이스

import sqlite3

conn = sqlite3.connect(":memory:")
cur = conn.cursor()

cur.execute("CREATE TABLE books (id INTEGER PRIMARY KEY, title TEXT, author TEXT)")
cur.execute("INSERT INTO books VALUES (1, '어린 왕자', '생텍쥐페리')")
cur.execute("INSERT INTO books VALUES (2, '데미안', '헤르만 헤세')")

cur.execute("SELECT title, author FROM books")
print(cur.fetchall())

conn.close()

도서관 데이터도서관 검색 시스템은 책 제목, 저자, 도서번호를 데이터베이스에 저장하고 필요할 때 조회합니다.

예제 2. 학생 점수 조건 검색

import sqlite3

예제 3. 매점 판매량 정렬

import sqlite3

    ("물", 35), ("빵", 42), ("우유", 19), ("주스", 28)

예제 4. 학년별 평균 점수

import sqlite3

conn = sqlite3.connect(":memory:")
cur = conn.cursor()

cur.executemany("INSERT INTO scores VALUES (?, ?)", [

Closing Talk

8. 수업을 마무리하는 멘트

오늘 우리는 데이터가 단순한 기록에서 출발해 정보와 지식으로 발전하고, 데이터베이스 안에서 체계적으로 관리된다는 것을 배웠습니다. 컴퓨터과학은 데이터베이스와 SQL을 통해 많은 데이터를 빠르게 저장하고 검색하게 해 줍니다. 수학은 수치형 데이터, 범주형 데이터, 평균과 집계 같은 분석 기준을 제공합니다. 과학은 관찰과 측정으로 신뢰할 수 있는 데이터를 만들고, 그 데이터를 근거로 설명과 예측을 수행합니다. 데이터베이스와 SQL은 데이터 과학과 머신러닝에서 데이터를 준비하고 이해하는 가장 중요한 기초 도구입니다.

컴퓨터과학과의 연결

데이터베이스, 테이블, SQL, 알고리즘을 통해 데이터를 효율적으로 저장·조회·관리합니다.

수학과의 연결

수치형·범주형 데이터 구분, 평균, 집계, 정렬, 비교는 수학적 사고와 연결됩니다.

과학과의 연결

관찰과 실험으로 얻은 데이터를 데이터베이스에 저장하고 분석하여 가설을 검증합니다.

머신러닝과의 연결

머신러닝 모델 학습 전에는 SQL로 필요한 데이터를 정확히 추출하고 정리하는 과정이 필요합니다.
