Opening Talk

2. 수업을 여는 멘트

여러분은 날씨 앱, 주식 차트, 게임 랭킹, 유튜브 조회수 그래프, 학교 성적표의 막대그래프를 본 적이 있을 것입니다. 같은 데이터라도 숫자로 길게 나열하면 이해하기 어렵지만, 그래프로 표현하면 변화와 차이가 한눈에 보입니다. 오늘 수업에서는 데이터를 보기 좋게 꾸미는 것을 넘어, 정확하게 보여 주고 올바르게 해석하는 방법을 배웁니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 데이터 시각화

데이터 시각화는 숫자, 표, 기록으로 된 데이터를 그래프나 차트로 표현하여 패턴과 관계를 쉽게 이해하도록 만드는 과정입니다. 시각화는 분석 결과를 전달하는 언어이며, 복잡한 데이터를 빠르게 파악하게 해 줍니다.

개념 2. 그래프 선택

좋은 시각화는 예쁜 그래프가 아니라 목적에 맞는 그래프입니다. 항목 비교는 막대그래프, 시간 변화는 선그래프, 분포는 히스토그램, 두 변수의 관계는 산점도가 적절합니다.

개념 3. 막대그래프

막대그래프는 여러 항목의 크기를 비교할 때 사용합니다. 예를 들어 과목별 평균 점수, 상품별 판매량, 지역별 인구를 비교할 수 있습니다.

개념 4. 선그래프

선그래프는 시간에 따른 변화를 보여 줄 때 유용합니다. 기온 변화, 주가 변화, 월별 매출, 일별 걸음 수처럼 순서가 있는 데이터에 적합합니다.

개념 5. 히스토그램

히스토그램은 수치형 데이터가 어떤 구간에 얼마나 많이 모여 있는지를 보여 줍니다. 시험 점수 분포, 키 분포, 대기 시간 분포를 분석할 때 사용합니다.

개념 6. 산점도

산점도는 두 변수 사이의 관계를 점으로 나타냅니다. 공부 시간과 점수, 기온과 아이스크림 판매량, 운동 시간과 심박수의 관계를 볼 수 있습니다.

개념 7. 축, 단위, 범례

그래프를 해석할 때는 제목, x축, y축, 단위, 범례를 확인해야 합니다. 축의 범위가 과도하게 좁거나 넓으면 차이가 실제보다 크게 또는 작게 보일 수 있습니다.

개념 8. 그래프 왜곡

그래프는 데이터를 쉽게 보여 주지만 잘못 만들면 오해를 만들 수 있습니다. y축을 0이 아닌 값에서 시작하거나, 단위를 숨기거나, 일부 데이터만 선택하면 왜곡된 결론이 나올 수 있습니다.

개념 9. 데이터 스토리텔링

데이터 스토리텔링은 그래프를 이용해 데이터가 말하는 핵심 메시지를 전달하는 과정입니다. 단순히 그래프를 만드는 것이 아니라 “그래서 무엇을 알 수 있는가?”를 설명해야 합니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 막대그래프: 항목별 비교

import matplotlib.pyplot as plt

items = ["물", "주스", "빵", "우유"]
sales = [35, 28, 42, 19]

plt.bar(items, sales)
plt.title("매점 상품별 판매량")
plt.xlabel("상품")
plt.ylabel("판매량")
plt.show()

설명: 막대그래프는 상품별 판매량처럼 항목 간 크기를 비교할 때 적합합니다.

코드 2. 선그래프: 시간에 따른 변화

import matplotlib.pyplot as plt

days = ["월", "화", "수", "목", "금"]
temperature = [22, 24, 26, 25, 28]

plt.plot(days, temperature, marker="o")
plt.title("요일별 기온 변화")
plt.xlabel("요일")
plt.ylabel("기온")
plt.show()

설명: 선그래프는 요일별 기온처럼 시간이나 순서에 따른 변화를 표현합니다.

코드 3. 히스토그램: 분포 확인

import matplotlib.pyplot as plt

scores = [55, 60, 62, 70, 75, 78, 80, 83, 88, 90, 95, 100]

plt.hist(scores, bins=5)
plt.title("학급 점수 분포")
plt.xlabel("점수 구간")
plt.ylabel("학생 수")
plt.show()

설명: 히스토그램은 점수가 어느 구간에 많이 모여 있는지 보여 줍니다.

코드 4. 산점도: 두 변수의 관계

import matplotlib.pyplot as plt

study_hours = [1, 2, 3, 4, 5, 6]
scores = [55, 62, 70, 78, 86, 92]

plt.scatter(study_hours, scores)
plt.title("공부 시간과 시험 점수")
plt.xlabel("공부 시간")
plt.ylabel("시험 점수")
plt.show()

설명: 산점도는 공부 시간이 늘어날수록 점수가 어떻게 변하는지 관계를 확인할 때 사용합니다.

코드 5. 원그래프: 전체에서 차지하는 비율

import matplotlib.pyplot as plt

labels = ["수면", "학교", "공부", "여가"]
hours = [8, 7, 4, 5]

plt.pie(hours, labels=labels, autopct="%1.1f%%")
plt.title("하루 시간 사용 비율")
plt.show()

설명: 원그래프는 전체 중 각 항목이 차지하는 비율을 보여 줄 때 사용합니다.

코드 6. 제목·축·범례 넣기

import matplotlib.pyplot as plt

months = ["3월", "4월", "5월", "6월"]
class_a = [75, 78, 82, 85]
class_b = [72, 76, 79, 83]

plt.plot(months, class_a, label="A반")
plt.plot(months, class_b, label="B반")
plt.title("월별 평균 점수 변화")
plt.xlabel("월")
plt.ylabel("평균 점수")
plt.legend()
plt.show()

설명: 범례는 여러 데이터가 한 그래프에 있을 때 각각이 무엇을 의미하는지 알려 줍니다.
