Opening Talk

2. 수업을 여는 멘트

여러분이 학급 매점에서 어떤 음료를 더 준비해야 할지 결정한다고 생각해 봅시다. “내가 좋아하는 음료”를 기준으로 고르면 친구들의 실제 선택과 다를 수 있습니다. 하지만 일주일 동안 음료별 판매량을 기록하고, 가장 많이 팔린 음료, 요일별 변화, 날씨와 판매량의 관계를 분석하면 더 합리적인 결정을 할 수 있습니다. 오늘 수업에서는 단순한 숫자 목록을 의미 있는 결론으로 바꾸는 과정, 즉 데이터 분석을 배웁니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 데이터 분석

데이터 분석은 수집된 데이터를 정리하고 계산하며 시각화하여 데이터 속의 패턴, 차이, 관계, 변화, 문제점을 찾아내는 과정입니다. 분석의 목적은 단순히 숫자를 계산하는 것이 아니라 “왜 이런 결과가 나왔는가?”, “앞으로 어떻게 행동해야 하는가?”에 답하는 것입니다.

개념 2. 탐색적 데이터 분석(EDA)

탐색적 데이터 분석은 본격적인 결론을 내리기 전에 데이터를 여러 각도에서 살펴보는 과정입니다. 데이터의 크기, 최솟값과 최댓값, 평균, 분포, 이상치, 빠진 값 등을 확인합니다.

개념 3. 대표값

대표값은 많은 데이터를 하나의 값으로 요약하는 방법입니다. 평균은 전체 합을 자료의 개수로 나눈 값이고, 중앙값은 데이터를 순서대로 정렬했을 때 가운데에 있는 값입니다.

\[\text{평균}=\frac{\text{자료의 총합}}{\text{자료의 개수}}\]

개념 4. 산포도와 범위

산포도는 데이터가 얼마나 흩어져 있는지를 나타냅니다. 가장 간단한 산포도 지표는 범위이며, 범위는 최댓값에서 최솟값을 뺀 값입니다.

\[\text{범위}=\text{최댓값}-\text{최솟값}\]

개념 5. 분포

분포는 데이터가 어떤 값 주변에 얼마나 많이 모여 있는지 보여 줍니다. 히스토그램은 분포를 확인할 때 자주 사용됩니다.

개념 6. 이상치

이상치는 다른 값들과 매우 동떨어진 데이터입니다. 이상치는 오류일 수도 있고 특별한 사건의 신호일 수도 있으므로 원인을 확인해야 합니다.

개념 7. 상관관계

상관관계는 두 변수 사이에 함께 변하는 경향이 있는지를 보는 개념입니다. 단, 상관관계가 있다고 해서 반드시 인과관계가 있다는 뜻은 아닙니다.

중요한 주의: 상관관계는 인과관계를 의미하지 않습니다.

개념 8. 데이터 시각화

데이터 시각화는 숫자로 된 데이터를 그래프와 차트로 표현하는 방법입니다. 막대그래프는 항목 비교, 선그래프는 시간 변화, 히스토그램은 분포, 산점도는 두 변수의 관계 파악에 유용합니다.

개념 9. 데이터 해석과 의사결정

분석 결과는 의사결정으로 이어져야 합니다. 예를 들어 “금요일에 매점 판매량이 높다”는 분석 결과가 나오면 금요일 재고를 늘리는 결정을 할 수 있습니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 데이터 확인

sales = [35, 42, 38, 50, 61, 58, 45]
print("데이터 개수:", len(sales))
print("판매량 데이터:", sales)

설명: 분석은 먼저 데이터의 개수와 형태를 확인하는 것에서 시작합니다.

코드 2. 평균 계산하기

scores = [80, 90, 75, 85, 100]
average = sum(scores) / len(scores)
print("평균 점수:", average)

설명: 평균은 데이터를 하나의 대표값으로 요약합니다.

코드 3. 중앙값 계산하기

data = [100, 70, 80, 90, 60]
data.sort()
middle = len(data) // 2
median = data[middle]
print("정렬된 데이터:", data)
print("중앙값:", median)

설명: 중앙값은 데이터를 정렬했을 때 가운데 있는 값입니다.

코드 4. 범위 구하기

temperatures = [22, 24, 27, 29, 31, 28, 25]
data_range = max(temperatures) - min(temperatures)
print("최고 기온:", max(temperatures))
print("최저 기온:", min(temperatures))
print("기온 범위:", data_range)

설명: 범위는 데이터가 얼마나 넓게 퍼져 있는지 간단히 보여 줍니다.

코드 5. 이상치 찾기

steps = [5200, 6100, 7300, 6900, 8100, 100000, 7600]
for s in steps:
    if s > 30000:
        print("이상치 의심:", s)
    else:
        print("정상 범위:", s)

설명: 대부분의 값과 크게 다른 값은 오류인지 특별한 사건인지 확인해야 합니다.

코드 6. 막대그래프

import matplotlib.pyplot as plt
items = ["물", "주스", "빵", "우유"]
sales = [35, 28, 42, 19]
plt.bar(items, sales)
plt.title("매점 상품별 판매량")
plt.xlabel("상품")
plt.ylabel("판매량")
plt.show()

설명: 막대그래프는 여러 항목의 크기를 비교할 때 유용합니다.

코드 7. 산점도

import matplotlib.pyplot as plt
temperature = [20, 22, 25, 27, 30, 32]
icecream_sales = [30, 35, 42, 50, 65, 72]
plt.scatter(temperature, icecream_sales)
plt.title("기온과 아이스크림 판매량")
plt.xlabel("기온")
plt.ylabel("아이스크림 판매량")
plt.show()

설명: 산점도는 두 변수 사이의 관계를 눈으로 확인할 때 사용합니다.

1. 데이터 분석은 데이터 속에서 의미 있는 패턴과 관계를 찾는 과정이다. (O/X)

단순 계산이 아니라 패턴, 차이, 관계를 찾아 해석하는 과정입니다.

정답: X
극단적인 값이 있으면 평균이 왜곡될 수 있습니다.

3. 중앙값은 데이터를 정렬했을 때 가운데 있는 값이다. (O/X)

정답: O
작은 값부터 큰 값까지 정렬했을 때 중앙에 위치한 값입니다.

4. 범위는 최솟값에서 최댓값을 뺀 값이다. (O/X)

정답: X
범위는 최댓값에서 최솟값을 뺀 값입니다.

6. 형성평가 5지선다형 10문항 + 상세풀이

1. 데이터 분석의 가장 적절한 설명은?

  • ① 데이터를 무조건 삭제하는 과정
  • ② 데이터를 저장만 하는 과정
  • ③ 데이터 속의 패턴과 의미를 찾는 과정
  • ④ 데이터를 암호로 바꾸는 과정
  • ⑤ 그래프를 보지 않는 과정
정답: ③
데이터 분석은 데이터에서 의미 있는 특징과 결론을 찾아내는 과정입니다.

2. 평균을 구하는 식으로 옳은 것은?

  • ① 최댓값 - 최솟값
  • ② 자료의 총합 / 자료의 개수
  • ③ 자료의 개수 / 자료의 총합
  • ④ 최댓값 + 최솟값
  • ⑤ 가장 자주 나온 값만 선택
정답: ②
평균은 모든 자료를 더한 뒤 자료의 개수로 나눕니다.

3. 데이터가 [10, 20, 30, 40, 100]일 때 중앙값은?

  • ① 10
  • ② 20
  • ③ 30
  • ④ 40
  • ⑤ 100
정답: ③
정렬된 데이터에서 가운데 값은 30입니다.

4. 데이터 [3, 7, 10, 15]의 범위는?

  • ① 3
  • ② 7
  • ③ 10
  • ④ 12
  • ⑤ 15
정답: ④
15 - 3 = 12입니다.

5. 시간에 따른 온도 변화를 나타내기에 가장 적절한 그래프는?

  • ① 선그래프

  • ③ 산점도만
  • ④ 그림 파일
  • ⑤ 코드 주석
정답: ①
시간 변화는 선그래프로 표현하면 흐름을 잘 볼 수 있습니다.

6. 항목별 판매량 비교에 가장 적절한 그래프는?

  • ① 막대그래프
  • ② 선그래프
  • ③ 히스토그램만
  • ④ 지도
  • ⑤ 텍스트 문단
정답: ①
막대그래프는 여러 항목의 크기를 비교하는 데 적합합니다.

7. 이상치에 대한 설명으로 옳은 것은?

  • ① 항상 가장 좋은 데이터이다.
  • ② 다른 값들과 매우 동떨어진 값이다.
  • ③ 절대 확인할 필요가 없다.
  • ④ 모든 데이터의 평균이다.
  • ⑤ 그래프에서 보이지 않는다.
정답: ②
이상치는 다른 값들과 크게 다른 값입니다.

8. 두 변수 사이의 관계를 보기 위해 주로 사용하는 그래프는?

  • ① 산점도
  • ② 표지 그림
  • ③ 목차
  • ④ 코드 블록
  • ⑤ 글머리표
정답: ①
산점도는 두 변수의 관계를 점으로 나타냅니다.

9. 상관관계 해석에서 주의해야 할 점은?

  • ① 상관관계는 항상 인과관계이다.
  • ② 상관관계는 인과관계를 반드시 의미하지 않는다.
  • ③ 상관관계는 그래프로 볼 수 없다.
  • ④ 두 변수는 함께 분석할 수 없다.
  • ⑤ 상관관계는 데이터 분석과 무관하다.
정답: ②
상관관계가 있어도 원인과 결과 관계라고 단정하면 안 됩니다.

10. 데이터 분석 결과를 해석할 때 가장 바람직한 태도는?

  • ① 그래프만 보고 즉시 단정한다.
  • ② 데이터의 출처와 맥락을 함께 확인한다.
  • ③ 이상치는 무조건 무시한다.
  • ④ 평균만 보고 모든 결론을 낸다.
  • ⑤ 분석 결과를 의사결정에 절대 사용하지 않는다.
정답: ②
출처, 수집 방법, 맥락과 함께 해석해야 신뢰할 수 있습니다.
Real Life Coding

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 일주일 공부 시간 분석

study_hours = [2, 3, 1, 4, 2, 5, 3]
average = sum(study_hours) / len(study_hours)
total = sum(study_hours)
print("총 공부 시간:", total)
print("하루 평균 공부 시간:", round(average, 2))

학습 데이터공부 시간 데이터를 분석하면 자신의 학습 습관을 객관적으로 확인하고 계획을 조정할 수 있습니다.

예제 2. 매점 판매량 최고 상품 찾기

sales = {"물":35, "주스":28, "빵":42, "우유":19, "과자":31}
best_item = max(sales, key=sales.get)
print("가장 많이 팔린 상품:", best_item)
print("판매량:", sales[best_item])

경영 데이터판매량을 분석하면 인기 상품을 파악하고 재고를 합리적으로 준비할 수 있습니다.

예제 3. 걸음 수 이상치 탐지

steps = [5200, 6100, 7300, 6900, 8100, 100000, 7600]
normal, outliers = [], []
for s in steps:
    if s > 30000:
        outliers.append(s)
    else:
        normal.append(s)
print("정상 데이터:", normal)
print("이상치:", outliers)

건강 데이터비정상적으로 큰 값은 기기 오류인지 특별한 활동인지 확인해야 합니다.

예제 4. 날씨와 아이스크림 판매량 관계 보기

import matplotlib.pyplot as plt
temperature = [20, 22, 25, 27, 30, 32]
sales = [30, 35, 42, 50, 65, 72]
plt.scatter(temperature, sales)
plt.title("기온과 아이스크림 판매량 관계")
plt.xlabel("기온")
plt.ylabel("판매량")
plt.show()

상관관계기온이 높아질수록 판매량도 증가하는 경향이 보이면 양의 상관관계를 생각할 수 있습니다.

Closing Talk

8. 수업을 마무리하는 멘트

오늘 우리는 데이터 분석이 단순히 평균을 구하는 활동이 아니라, 현실 세계에서 발생한 자료를 정리하고, 비교하고, 시각화하며, 의미 있는 결론을 찾는 과정임을 배웠습니다. 컴퓨터과학은 데이터를 저장하고 빠르게 처리하는 도구를 제공하고, 수학은 평균·중앙값·범위·분포·상관관계처럼 데이터를 해석하는 언어를 제공합니다. 과학은 관찰과 실험을 통해 데이터를 만들고, 분석 결과를 근거로 가설을 검증하게 해 줍니다. 데이터 분석은 컴퓨터과학, 수학, 과학이 만나는 지점이며, 머신러닝으로 나아가기 위한 가장 중요한 기초입니다.

컴퓨터과학과의 연결

파이썬, 리스트, 딕셔너리, 반복문, 시각화 라이브러리를 활용하여 데이터를 처리합니다.

수학과의 연결

평균, 중앙값, 범위, 분포, 상관관계를 통해 데이터의 특징을 정량적으로 해석합니다.

과학과의 연결

관찰, 측정, 실험, 검증 과정을 통해 데이터 분석 결과의 신뢰성을 판단합니다.

머신러닝과의 연결

데이터 분석을 통해 입력 데이터의 특징을 이해하면 더 좋은 예측 모델을 만들 수 있습니다.
