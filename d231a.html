Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 모델 성능

모델 성능은 머신러닝 모델이 실제값을 얼마나 잘 예측하는지를 나타내는 정도입니다. 모델의 성능은 느낌이 아니라 실제값과 예측값을 비교한 수치로 평가해야 합니다.

개념 2. 분류 모델과 회귀 모델

분류 모델은 스팸/정상, 합격/불합격처럼 범주를 예측합니다. 회귀 모델은 집값, 점수, 기온처럼 연속적인 숫자를 예측합니다. 모델 종류에 따라 성능 지표도 달라집니다.

개념 3. 혼동행렬

혼동행렬은 분류 모델의 예측 결과를 실제값과 비교하여 맞힌 경우와 틀린 경우를 표로 정리한 것입니다. TP, TN, FP, FN 네 가지 요소로 구성됩니다.

예측 양성예측 음성
실제 양성TPFN
실제 음성FPTN

개념 4. 정확도

정확도는 전체 예측 중 맞힌 비율입니다. 직관적이지만 데이터가 불균형할 때는 모델 성능을 과대평가할 수 있습니다.

\[ \text{정확도} = \frac{TP+TN}{TP+TN+FP+FN} \]

개념 5. 정밀도

정밀도는 모델이 양성이라고 예측한 것 중 실제 양성의 비율입니다.

\[ \text{정밀도} = \frac{TP}{TP+FP} \]

개념 6. 재현율

재현율은 실제 양성 중 모델이 양성으로 잘 찾아낸 비율입니다. 질병 진단처럼 실제 환자를 놓치지 않는 것이 중요한 경우 재현율이 중요합니다.

\[ \text{재현율} = \frac{TP}{TP+FN} \]

개념 7. F1-score

F1-score는 정밀도와 재현율의 조화평균입니다. 정밀도와 재현율을 균형 있게 고려할 때 사용합니다.

\[ F1 = 2 \cdot \frac{\text{정밀도}\cdot\text{재현율}}{\text{정밀도}+\text{재현율}} \]

개념 8. 회귀 모델의 오차

회귀 모델은 숫자를 예측하므로 실제값과 예측값의 차이, 즉 오차를 계산합니다. 오차가 작을수록 예측이 실제값에 가깝습니다.

개념 9. MAE, MSE, RMSE

MAE는 절대오차의 평균, MSE는 제곱오차의 평균, RMSE는 MSE의 제곱근입니다. RMSE는 큰 오차에 더 민감합니다.

\[ MAE=\frac{1}{n}\sum |y_i-\hat{y_i}| \]

\[ MSE=\frac{1}{n}\sum (y_i-\hat{y_i})^2 \]

개념 10. 데이터 불균형과 지표 선택

데이터가 한쪽 범주에 매우 치우쳐 있으면 정확도만으로 모델을 평가하기 어렵습니다. 실제 환자가 1%뿐인 데이터에서 모두 “정상”이라고 예측해도 정확도는 99%가 될 수 있지만 좋은 진단 모델은 아닙니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 실제값과 예측값 비교

y_true = [1, 0, 1, 1, 0, 0, 1]
y_pred = [1, 0, 1, 0, 0, 1, 1]

for real, pred in zip(y_true, y_pred):
    print("실제값:", real, "예측값:", pred)

설명: 모델 성능 평가는 실제값과 예측값을 비교하는 것에서 시작합니다.

코드 2. 혼동행렬 요소 계산

y_true = [1, 0, 1, 1, 0, 0, 1]
y_pred = [1, 0, 1, 0, 0, 1, 1]

TP = TN = FP = FN = 0

for real, pred in zip(y_true, y_pred):
    if real == 1 and pred == 1:
        TP += 1
    elif real == 0 and pred == 0:
        TN += 1
    elif real == 0 and pred == 1:
        FP += 1
    elif real == 1 and pred == 0:
        FN += 1

print("TP:", TP, "TN:", TN, "FP:", FP, "FN:", FN)

설명: TP, TN, FP, FN을 계산하면 정확도, 정밀도, 재현율을 구할 수 있습니다.

코드 3. 정확도 계산

TP, TN, FP, FN = 3, 2, 1, 1

accuracy = (TP + TN) / (TP + TN + FP + FN)

print("정확도:", accuracy)

설명: 전체 예측 중 맞힌 비율을 계산합니다.

코드 4. 정밀도와 재현율 계산

TP, TN, FP, FN = 3, 2, 1, 1

precision = TP / (TP + FP)
recall = TP / (TP + FN)

print("정밀도:", precision)
print("재현율:", recall)

설명: 정밀도는 예측 양성의 정확성, 재현율은 실제 양성을 찾아낸 비율입니다.

코드 5. F1-score 계산

precision = 0.75
recall = 0.75

f1 = 2 * precision * recall / (precision + recall)

print("F1-score:", f1)

설명: F1-score는 정밀도와 재현율을 균형 있게 반영합니다.

코드 6. 회귀 모델 오차 계산

y_true = [80, 90, 75, 100]
y_pred = [82, 88, 78, 95]

errors = [real - pred for real, pred in zip(y_true, y_pred)]

print("오차:", errors)

설명: 회귀 모델에서는 실제값과 예측값의 차이를 오차로 계산합니다.

코드 7. MAE, MSE, RMSE 계산

y_true = [80, 90, 75, 100]
y_pred = [82, 88, 78, 95]

errors = [real - pred for real, pred in zip(y_true, y_pred)]

mae = sum(abs(e) for e in errors) / len(errors)
mse = sum(e ** 2 for e in errors) / len(errors)
rmse = mse ** 0.5

print("MAE:", mae)
print("MSE:", mse)
print("RMSE:", rmse)

설명: MAE는 평균적인 오차 크기, MSE와 RMSE는 큰 오차에 더 민감한 지표입니다.

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 스팸 메일 분류 정확도

y_true = [1, 0, 1, 0, 1, 0]
y_pred = [1, 0, 0, 0, 1, 1]

correct = sum(real == pred for real, pred in zip(y_true, y_pred))
accuracy = correct / len(y_true)

print("정확도:", accuracy)

스팸 분류스팸과 정상 메일을 얼마나 맞혔는지 전체 정확도로 확인할 수 있습니다.

예제 2. 질병 진단 재현율

TP = 45
FN = 5

recall = TP / (TP + FN)

print("재현율:", recall)

의료 데이터실제 환자 중 모델이 환자로 찾아낸 비율입니다. 질병 진단에서는 재현율이 매우 중요할 수 있습니다.

예제 3. 스팸 판정 정밀도

TP = 30
FP = 10

precision = TP / (TP + FP)

print("정밀도:", precision)

메일 필터링스팸이라고 예측한 메일 중 실제 스팸의 비율입니다.

예제 4. 집값 예측 MAE

real_price = [3.0, 4.2, 5.1, 6.0]
pred_price = [3.2, 4.0, 4.8, 6.5]

mae = sum(abs(r - p) for r, p in zip(real_price, pred_price)) / len(real_price)

print("MAE:", mae)

부동산 예측예측 가격이 실제 가격과 평균적으로 얼마나 차이 나는지 확인할 수 있습니다.

예제 5. 시험 점수 예측 RMSE

real = [80, 90, 75, 100]
pred = [82, 88, 78, 95]

mse = sum((r - p) ** 2 for r, p in zip(real, pred)) / len(real)
rmse = mse ** 0.5

print("RMSE:", rmse)

교육 예측RMSE는 큰 예측 오차를 더 크게 반영하므로 예측이 크게 빗나간 경우를 확인하는 데 유용합니다.
