Opening Talk

2. 수업을 여는 멘트

뉴스에서 “전국 성인 1,000명을 대상으로 조사한 결과 찬성 52%, 오차범위 ±3.1%p”라는 표현을 본 적이 있을 것입니다. 전국 모든 사람에게 물어보지 않았는데 어떻게 전체 의견을 말할 수 있을까요? 핵심은 표본입니다. 전체를 모두 조사하기 어렵기 때문에 일부를 뽑아 전체를 추정합니다. 오늘은 표본을 이용해 전체를 추정하는 통계적 추정의 원리를 배우고, 그 결과가 얼마나 불확실한지도 함께 해석해 보겠습니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 모집단

모집단은 우리가 알고 싶은 전체 대상입니다. 예를 들어 “우리 학교 전체 학생의 평균 수면 시간”을 알고 싶다면 우리 학교 전체 학생이 모집단입니다.

개념 2. 표본

표본은 모집단에서 실제로 조사한 일부 대상입니다. 전체를 모두 조사하기 어렵거나 시간이 많이 걸릴 때 표본을 사용합니다.

개념 3. 표본추출

표본추출은 모집단에서 표본을 뽑는 과정입니다. 표본이 한쪽으로 치우치면 전체를 잘 대표하지 못하므로 무작위 추출과 대표성이 중요합니다.

개념 4. 모수와 통계량

모수는 모집단 전체의 특성값이고, 통계량은 표본에서 계산한 값입니다. 모집단 평균은 모수이고, 표본평균은 통계량입니다.

개념 5. 점추정

점추정은 하나의 값으로 모집단의 특성을 추정하는 방법입니다. 예를 들어 표본평균 7.2시간으로 전체 학생의 평균 수면 시간을 추정하는 것입니다.

개념 6. 구간추정

구간추정은 하나의 값이 아니라 범위로 추정하는 방법입니다. 예를 들어 “전체 평균 수면 시간은 6.9시간에서 7.5시간 사이일 것으로 추정된다”라고 말하는 방식입니다.

개념 7. 표준오차

표준오차는 표본통계량이 표본마다 얼마나 흔들릴 수 있는지를 나타내는 값입니다. 표본의 크기가 커질수록 표준오차는 일반적으로 작아집니다.

\[ SE = \frac{s}{\sqrt{n}} \]

개념 8. 오차범위

오차범위는 추정값 주변에 어느 정도의 불확실성이 있는지를 나타냅니다. 여론조사에서 ±3%p라고 하면 실제 비율이 조사 결과보다 약간 높거나 낮을 수 있음을 뜻합니다.

개념 9. 신뢰구간

신뢰구간은 추정값과 오차범위를 이용해 만든 추정 범위입니다. 고등학교 수준에서는 보통 “추정값 ± 오차범위” 형태로 이해할 수 있습니다.

\[ \text{신뢰구간} = \text{추정값} \pm \text{오차범위} \]

개념 10. 표본추출 편향

표본추출 편향은 표본이 모집단을 고르게 대표하지 못할 때 생깁니다. 예를 들어 온라인 설문만 하면 인터넷을 자주 쓰지 않는 사람들의 의견이 빠질 수 있습니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 모집단에서 표본 뽑기

import random

population = [160, 165, 170, 172, 168, 174, 169, 171, 166, 175]
sample = random.sample(population, 4)

print("모집단:", population)
print("표본:", sample)

설명: 전체 학생 키 데이터에서 일부 표본을 무작위로 뽑는 예제입니다.

코드 2. 표본평균으로 모집단 평균 추정하기

sample = [165, 170, 172, 168]

sample_mean = sum(sample) / len(sample)

print("표본평균:", sample_mean)
print("모집단 평균의 추정값:", sample_mean)

설명: 표본평균은 모집단 평균을 추정하는 대표적인 통계량입니다.

코드 3. 표본이 달라지면 추정값도 달라짐

import random

population = [160, 165, 170, 172, 168, 174, 169, 171, 166, 175]

for i in range(5):
    sample = random.sample(population, 4)
    mean = sum(sample) / len(sample)
    print(i+1, "번째 표본:", sample, "표본평균:", mean)

설명: 표본을 어떻게 뽑느냐에 따라 표본평균이 조금씩 달라질 수 있습니다.

코드 4. 표준오차 계산하기

sample = [165, 170, 172, 168, 171]

mean = sum(sample) / len(sample)
variance = sum((x - mean) ** 2 for x in sample) / (len(sample) - 1)
sample_std = variance ** 0.5
standard_error = sample_std / (len(sample) ** 0.5)

print("표본평균:", mean)
print("표본표준편차:", sample_std)
print("표준오차:", standard_error)

설명: 표준오차는 표본평균이 표본마다 얼마나 흔들릴 수 있는지 나타냅니다.

코드 5. 95% 신뢰구간 간단 계산

sample_mean = 169.2
standard_error = 1.2

margin_error = 1.96 * standard_error
lower = sample_mean - margin_error
upper = sample_mean + margin_error

print("오차범위:", margin_error)
print("95% 신뢰구간:", lower, "~", upper)

설명: 95% 신뢰구간은 표본평균 주변에 오차범위를 더하고 빼서 구할 수 있습니다.

코드 6. 표본 크기와 표준오차 비교

sample_std = 10

for n in [25, 100, 400]:
    se = sample_std / (n ** 0.5)
    print("표본 크기:", n, "표준오차:", se)

설명: 표본 크기 \(n\)이 커질수록 표준오차는 작아지는 경향이 있습니다.

코드 7. 표본비율 추정

responses = ["찬성", "반대", "찬성", "찬성", "반대", "찬성"]

agree_count = responses.count("찬성")
sample_ratio = agree_count / len(responses)

print("찬성 표본비율:", sample_ratio)

설명: 표본에서 찬성한 비율을 이용해 모집단의 찬성 비율을 추정할 수 있습니다.

Real Life Coding

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 학교 수면 시간 표본평균 추정

sleep_hours = [6.5, 7.0, 5.5, 8.0, 7.5, 6.0, 7.2]

sample_mean = sum(sleep_hours) / len(sleep_hours)

print("표본 평균 수면 시간:", round(sample_mean, 2))

생활 데이터일부 학생의 수면 시간을 조사하여 전체 학생의 평균 수면 시간을 추정할 수 있습니다.

예제 2. 만족도 조사 찬성 비율 추정

responses = ["만족", "만족", "보통", "만족", "불만", "만족"]

satisfied = responses.count("만족")
ratio = satisfied / len(responses)

print("만족 표본비율:", round(ratio, 2))

설문 데이터표본에서 만족한 사람의 비율을 계산해 전체 만족 비율을 추정하는 출발점으로 사용할 수 있습니다.

예제 3. 무작위 표본추출 실습

import random

population = list(range(1, 101))
sample = random.sample(population, 10)

print("100명 중 뽑힌 표본:", sample)

표본추출무작위 표본추출은 특정 대상만 선택되는 편향을 줄이는 데 도움이 됩니다.

예제 4. 신뢰구간 계산

sample_mean = 82
standard_error = 2.5

margin_error = 1.96 * standard_error
lower = sample_mean - margin_error
upper = sample_mean + margin_error

print("95% 신뢰구간:", round(lower, 2), "~", round(upper, 2))

성적 데이터표본평균만 제시하는 것보다 신뢰구간을 함께 제시하면 추정의 불확실성을 표현할 수 있습니다.

예제 5. 표본 크기에 따른 오차 감소

sample_std = 12

for n in [25, 50, 100, 400]:
    se = sample_std / (n ** 0.5)
    print("표본 크기:", n, "표준오차:", round(se, 2))

조사 설계표본 크기가 커질수록 표준오차가 작아지는 경향이 있어 더 안정적인 추정이 가능합니다.
