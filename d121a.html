Opening Talk

2. 수업을 여는 멘트

여러분이 날씨 앱을 만들거나, 학급 점수 데이터를 분석하거나, 게임 캐릭터의 능력치를 계산하는 프로그램을 만든다고 생각해 봅시다. 코드는 어디에 작성하고, 어떻게 실행하며, 오류가 생기면 어디에서 확인할까요? 단순한 메모장만으로도 코드를 쓸 수는 있지만, 실제 개발과 데이터 분석에서는 코드 작성, 실행, 결과 확인, 오류 수정이 한곳에서 이루어지는 통합 개발 환경이 필요합니다. 오늘은 데이터 분석의 출발점이 되는 파이썬 실행 환경을 배웁니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 통합 개발 환경

통합 개발 환경은 코드를 작성하고, 실행하고, 오류를 찾고, 파일을 관리하는 기능을 한곳에 모은 프로그램입니다. 영어로는 IDE라고 하며, Integrated Development Environment의 줄임말입니다.

개념 2. 코드 편집기

코드 편집기는 프로그램 코드를 입력하고 수정하는 공간입니다. 문법 강조, 자동 완성, 줄 번호, 들여쓰기 지원 기능이 있어 코드를 더 쉽게 작성하게 도와줍니다.

개념 3. 인터프리터

파이썬 인터프리터는 작성한 파이썬 코드를 한 줄씩 읽고 실행하는 프로그램입니다. 사람이 쓴 코드를 컴퓨터가 실행할 수 있도록 해석하는 역할을 합니다.

개념 4. 콘솔과 터미널

콘솔 또는 터미널은 프로그램 실행 결과가 출력되고, 사용자가 명령을 입력할 수 있는 공간입니다. print()의 결과나 오류 메시지를 확인하는 곳입니다.

개념 5. 셀 실행

Jupyter Notebook과 Google Colab은 코드를 셀 단위로 나누어 실행할 수 있습니다. 데이터 분석에서는 전체 코드를 한 번에 실행하기보다 데이터 확인, 계산, 그래프 생성 과정을 나누어 실행하는 경우가 많습니다.

개념 6. 디버깅

디버깅은 코드에서 오류를 찾고 수정하는 과정입니다. 오류 메시지는 문제의 위치와 원인을 알려 주는 힌트이므로, 무시하지 말고 차근차근 읽어야 합니다.

개념 7. 파일 저장과 실행

파이썬 코드는 보통 .py 확장자로 저장합니다. Jupyter Notebook은 .ipynb 형식을 사용합니다. 파일 형식에 따라 실행 방식과 사용 환경이 달라질 수 있습니다.

개념 8. 라이브러리와 패키지

라이브러리는 자주 사용하는 기능을 모아 둔 코드 묶음입니다. 데이터 분석에서는 pandas, matplotlib, numpy 같은 라이브러리를 자주 사용합니다.

개념 9. 데이터 분석 환경

데이터 분석 환경은 코드 실행뿐 아니라 데이터 파일 불러오기, 계산 결과 확인, 그래프 출력, 보고서 작성까지 지원해야 합니다. 그래서 Jupyter Notebook과 Google Colab이 데이터 분석 수업에서 많이 사용됩니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. IDE에서 기본 출력 확인

print("Hello, IDE!")
print("데이터 분석을 시작합니다.")

설명: IDE에서 가장 먼저 확인할 것은 코드가 정상적으로 실행되고 결과가 출력되는지입니다.

코드 2. 변수와 계산 실행

a = 10
b = 20
result = a + b

print("계산 결과:", result)

설명: 코드 편집기에 변수를 작성하고 실행하면 콘솔에서 계산 결과를 확인할 수 있습니다.

코드 3. 오류 메시지 확인

print(score)

설명: score 변수를 만들지 않고 출력하면 NameError가 발생합니다. 오류 메시지는 어떤 이름이 정의되지 않았는지 알려 줍니다.

코드 4. 오류 수정하기

score = 90
print(score)

설명: 변수를 먼저 정의한 뒤 출력하면 오류가 해결됩니다. 디버깅은 이런 식으로 원인을 찾고 수정하는 과정입니다.

코드 5. 입력과 출력

name = input("이름을 입력하세요: ")
print(name, "님, 환영합니다!")

설명: IDE의 콘솔에서는 사용자가 값을 입력하고 프로그램이 그 결과를 처리할 수 있습니다.

코드 6. 데이터 분석 미니 코드

scores = [80, 90, 75, 88, 92]

average = sum(scores) / len(scores)

print("점수 데이터:", scores)
print("평균 점수:", average)

설명: 데이터 분석 환경에서는 리스트에 저장된 데이터를 계산하고 결과를 바로 확인할 수 있습니다.

코드 7. 라이브러리 사용 확인

import math

number = 16
print("제곱근:", math.sqrt(number))

설명: 라이브러리를 불러오면 직접 만들지 않아도 다양한 기능을 사용할 수 있습니다.

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 간단 계산기

a = int(input("첫 번째 숫자: "))
b = int(input("두 번째 숫자: "))

print("합:", a + b)
print("곱:", a * b)

계산 자동화IDE 콘솔에서 값을 입력받고 결과를 바로 확인하며 프로그램의 입력·처리·출력 구조를 이해할 수 있습니다.

예제 2. 학급 평균 계산

scores = [80, 90, 75, 88, 92]

average = sum(scores) / len(scores)

print("학급 평균:", round(average, 2))

교육 데이터점수 목록을 코드로 처리하면 사람이 직접 계산하지 않아도 평균을 빠르게 구할 수 있습니다.

예제 3. 오류 수정 실습

# 잘못된 코드: print(total)
total = 100
print(total)

디버깅변수를 먼저 정의한 뒤 출력해야 오류가 발생하지 않습니다. 오류 수정 과정을 통해 실행 순서를 배울 수 있습니다.

예제 4. 기온에 따른 안내 문구

temperature = 31

if temperature >= 30:
    print("더운 날씨입니다. 물을 충분히 마시세요.")
else:
    print("활동하기 좋은 날씨입니다.")

날씨 데이터조건문을 이용하면 숫자 데이터를 생활에 필요한 정보로 바꿀 수 있습니다.

예제 5. 라이브러리로 제곱근 계산

import math

numbers = [4, 9, 16, 25]

for n in numbers:
    print(n, "의 제곱근:", math.sqrt(n))

수학 계산라이브러리를 사용하면 복잡한 수학 기능을 쉽게 활용할 수 있습니다.

Closing Talk

8. 수업을 마무리하는 멘트

오늘 우리는 통합 개발 환경이 단순히 코드를 입력하는 공간이 아니라, 코드를 작성하고 실행하고 오류를 찾고 결과를 확인하는 데이터 분석의 출발점임을 배웠습니다. 컴퓨터과학에서는 IDE를 통해 알고리즘을 실제 프로그램으로 구현하고, 수학에서는 계산과 분석을 코드로 자동화하며, 과학에서는 관찰과 실험 데이터를 처리하고 결과를 검증합니다. 앞으로 데이터 분석과 머신러닝을 배울 때 IDE는 여러분의 실험실이자 계산기이며, 문제 해결을 위한 도구가 될 것입니다.

컴퓨터과학과의 연결

IDE는 알고리즘을 코드로 작성하고 실행하며 오류를 수정하는 개발 환경입니다.

수학과의 연결

평균, 합계, 함수, 반복 계산을 파이썬 코드로 자동화할 수 있습니다.

과학과의 연결

실험과 관찰 데이터를 코드로 정리하고 분석하여 결과를 검증할 수 있습니다.

머신러닝과의 연결

머신러닝 모델도 IDE나 노트북 환경에서 데이터를 불러오고 학습 코드를 실행하며 결과를 확인합니다.
