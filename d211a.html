Opening Talk

2. 수업을 여는 멘트

여러분은 “우리 반 평균 점수”, “한국인의 평균 수면 시간”, “한 달 평균 스마트폰 사용 시간”이라는 표현을 자주 듣습니다. 평균은 많은 데이터를 한눈에 이해하게 도와주지만, 항상 모든 사람의 실제 모습을 정확히 보여 주지는 않습니다. 예를 들어 대부분 학생이 70~80점인데 한 명이 0점 또는 100점을 받으면 평균은 크게 달라질 수 있습니다. 오늘은 데이터를 대표하는 값이 무엇인지, 언제 평균을 쓰고 언제 중앙값이나 최빈값을 써야 하는지 배워 보겠습니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 대표값

대표값은 많은 데이터를 하나의 값으로 요약하여 데이터의 중심적인 특징을 보여 주는 값입니다. 대표값은 자료 전체를 빠르게 이해하게 해 주지만, 데이터의 모든 정보를 담지는 못합니다.

개념 2. 평균

평균은 모든 값을 더한 뒤 자료의 개수로 나눈 값입니다. 모든 데이터를 계산에 반영하므로 널리 사용되지만, 매우 큰 값이나 작은 값인 이상치에 민감합니다.

\[ \text{평균} = \frac{\text{자료의 총합}}{\text{자료의 개수}} \]

개념 3. 중앙값

중앙값은 데이터를 크기순으로 정렬했을 때 가운데에 위치한 값입니다. 이상치가 있어도 평균보다 덜 흔들리므로 소득, 집값, 대기 시간처럼 극단값이 있는 자료에서 유용합니다.

개념 4. 최빈값

최빈값은 데이터에서 가장 자주 나타나는 값입니다. 숫자뿐 아니라 좋아하는 과목, 교통수단, 혈액형처럼 범주형 데이터에서도 사용할 수 있습니다.

개념 5. 가중평균

가중평균은 값마다 중요도나 비중이 다를 때 사용하는 평균입니다. 예를 들어 수행평가 40%, 지필평가 60%처럼 평가 비율이 다르면 단순 평균보다 가중평균이 더 적절합니다.

\[ \text{가중평균} = \frac{\sum x_i w_i}{\sum w_i} \]

개념 6. 사분위수

사분위수는 데이터를 네 부분으로 나누는 기준값입니다. 제1사분위수(Q1), 중앙값(Q2), 제3사분위수(Q3)를 통해 데이터가 어떻게 퍼져 있는지 알 수 있습니다.

개념 7. 이상치

이상치는 다른 값들과 크게 떨어져 있는 값입니다. 이상치는 입력 오류일 수도 있고 특별한 사건의 결과일 수도 있습니다. 이상치가 있으면 평균이 실제 중심보다 크게 또는 작게 왜곡될 수 있습니다.

개념 8. 대표값의 선택

자료가 고르게 분포하면 평균이 적절할 수 있습니다. 극단값이 있으면 중앙값이 더 적절할 수 있습니다. 가장 흔한 선택이나 유형을 알고 싶을 때는 최빈값이 적절합니다.

개념 9. 데이터 탐색에서 대표값의 역할

대표값은 데이터 탐색의 출발점입니다. 그러나 대표값만으로는 분포, 산포도, 이상치, 편향을 모두 알 수 없으므로 그래프와 함께 해석하는 것이 좋습니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 평균 구하기

scores = [80, 90, 75, 85, 100]

average = sum(scores) / len(scores)

print("평균:", average)

설명: 모든 점수를 더한 뒤 개수로 나누어 평균을 구합니다.

코드 2. 중앙값 구하기

data = [100, 70, 80, 90, 60]

data.sort()
middle = len(data) // 2
median = data[middle]

print("정렬된 데이터:", data)
print("중앙값:", median)

설명: 데이터를 정렬한 뒤 가운데 위치의 값을 선택합니다.

코드 3. 자료 개수가 짝수일 때 중앙값

data = [60, 70, 80, 90]

data.sort()
n = len(data)
median = (data[n//2 - 1] + data[n//2]) / 2

print("중앙값:", median)

설명: 자료 개수가 짝수이면 가운데 두 값의 평균을 중앙값으로 사용합니다.

코드 4. 최빈값 구하기

from collections import Counter

transport = ["버스", "도보", "버스", "지하철", "버스", "자전거"]

counts = Counter(transport)
mode = counts.most_common(1)[0]

print("최빈값:", mode[0])
print("빈도:", mode[1])

설명: 최빈값은 가장 자주 등장하는 값입니다. 범주형 데이터에도 사용할 수 있습니다.

코드 5. 이상치가 평균에 미치는 영향

normal_scores = [70, 72, 75, 78, 80]
with_outlier = [70, 72, 75, 78, 200]

print("이상치 없는 평균:", sum(normal_scores) / len(normal_scores))
print("이상치 있는 평균:", sum(with_outlier) / len(with_outlier))

설명: 200이라는 극단값이 포함되면 평균이 크게 올라갑니다.

코드 6. 가중평균 구하기

performance = 85
written = 92

weighted_average = performance * 0.4 + written * 0.6

print("가중평균:", weighted_average)

설명: 수행평가 40%, 지필평가 60%처럼 비중이 다를 때 가중평균을 사용합니다.

코드 7. 평균·중앙값·최빈값 한 번에 비교하기

from statistics import mean, median, mode

data = [70, 75, 75, 80, 85, 100]

print("평균:", mean(data))
print("중앙값:", median(data))
print("최빈값:", mode(data))

설명: 같은 데이터라도 대표값의 종류에 따라 다른 특징을 보여 줍니다.

3. 평균은 이상치의 영향을 전혀 받지 않는다. (O/X)

정답: X
평균은 모든 값을 반영하므로 극단적인 값에 크게 영향을 받을 수 있습니다.

4. 중앙값은 데이터를 정렬했을 때 가운데 있는 값이다. (O/X)

자료를 크기순으로 정렬한 뒤 중앙에 있는 값이 중앙값입니다.

5. 자료 개수가 짝수일 때 중앙값은 가운데 두 값의 평균으로 구할 수 있다. (O/X)

가운데가 하나로 정해지지 않으므로 가운데 두 값의 평균을 사용합니다.

6. 최빈값은 가장 자주 나타나는 값이다. (O/X)

정답: O
빈도가 가장 높은 값이 최빈값입니다.

최빈값은 교통수단, 혈액형, 선호 과목 같은 범주형 데이터에 매우 유용합니다.

8. 가중평균은 모든 값의 중요도가 같을 때만 사용한다. (O/X)

정답: X
가중평균은 값마다 중요도나 비중이 다를 때 사용합니다.

대표값은 중심을 보여 주지만 분포, 산포도, 이상치까지 모두 보여 주지는 못합니다.

10. 데이터 탐색에서는 대표값과 함께 그래프도 함께 보는 것이 좋다. (O/X)

정답: O
그래프는 분포와 이상치를 시각적으로 확인하는 데 도움을 줍니다.

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 학급 평균 점수 구하기

scores = [82, 91, 75, 88, 96]

average = sum(scores) / len(scores)

print("학급 평균 점수:", round(average, 2))

교육 데이터여러 학생의 점수를 평균으로 요약하면 학급의 전체적인 성취 수준을 빠르게 파악할 수 있습니다.

예제 2. 스마트폰 사용 시간의 중앙값

hours = [2.1, 2.5, 3.0, 3.2, 9.5]

hours.sort()
median = hours[len(hours)//2]

print("중앙값:", median)

생활 습관 데이터9.5시간처럼 극단적으로 긴 사용 시간이 있을 때 평균보다 중앙값이 일반적인 사용 시간을 더 잘 나타낼 수 있습니다.

예제 3. 가장 많이 이용한 교통수단

from collections import Counter

transport = ["버스", "지하철", "버스", "도보", "버스", "자전거"]

mode = Counter(transport).most_common(1)[0]

print("가장 많이 이용한 교통수단:", mode[0])

범주형 데이터교통수단처럼 숫자가 아닌 데이터는 평균보다 최빈값으로 대표하는 것이 적절합니다.

예제 4. 수행평가와 지필평가 가중평균

performance = 88
written = 94

final_score = performance * 0.4 + written * 0.6

print("최종 점수:", final_score)

평가 데이터평가 비율이 다르면 단순 평균이 아니라 가중평균을 사용해야 실제 반영 비율을 정확히 표현할 수 있습니다.

예제 5. 이상치가 평균을 바꾸는 효과

prices = [3000, 3200, 3500, 3300, 20000]

average = sum(prices) / len(prices)
sorted_prices = sorted(prices)
median = sorted_prices[len(sorted_prices)//2]

print("평균 가격:", average)
print("중앙값 가격:", median)

소비 데이터대부분 가격이 3000원대인데 20000원 상품이 있으면 평균이 크게 올라갑니다. 이럴 때 중앙값도 함께 확인해야 합니다.

Closing Talk

8. 수업을 마무리하는 멘트

오늘 우리는 많은 데이터를 하나의 값으로 요약하는 대표값을 배웠습니다. 평균은 모든 데이터를 반영하지만 이상치에 민감하고, 중앙값은 극단값이 있을 때 안정적이며, 최빈값은 가장 흔한 선택이나 유형을 보여 줍니다. 컴퓨터과학은 파이썬으로 대표값을 빠르게 계산하게 해 주고, 수학은 평균·중앙값·최빈값의 원리를 설명합니다. 과학은 관찰과 측정으로 얻은 데이터를 대표값으로 요약하여 현상을 비교하고 판단하게 합니다. 데이터 과학과 머신러닝에서는 대표값을 통해 데이터의 첫인상을 파악하고 더 깊은 분석으로 나아갑니다.

컴퓨터과학과의 연결

파이썬 리스트, 함수, 라이브러리를 활용해 대표값을 자동 계산하고 데이터 탐색을 수행합니다.

수학과의 연결

평균, 중앙값, 최빈값, 가중평균은 통계적 사고와 수리적 계산의 핵심입니다.

과학과의 연결

실험 결과와 관찰 자료를 대표값으로 요약해 집단의 특성과 경향을 설명합니다.

머신러닝과의 연결

모델 학습 전 데이터 탐색 단계에서 대표값은 데이터의 중심 경향과 이상치를 파악하는 출발점입니다.
