Opening Talk

2. 수업을 여는 멘트

여러분이 공부한 시간과 시험 점수 데이터를 모았다고 생각해 봅시다. 공부 시간이 늘어날수록 점수가 오르는 경향이 있다면, 아직 시험을 보지 않은 학생의 공부 시간만 보고 예상 점수를 추측할 수 있을까요? 회귀분석은 이런 질문에 답하기 위한 도구입니다. 데이터 점들 사이를 가장 잘 설명하는 선을 찾고, 그 선을 이용해 새로운 값을 예측합니다. 오늘은 데이터 과학과 머신러닝에서 매우 중요한 예측의 기초, 회귀분석을 배웁니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 회귀분석

회귀분석은 하나 이상의 변수와 결과값 사이의 관계를 수학적 모델로 표현하고, 이를 이용해 결과를 예측하는 방법입니다. 고등학교 수준에서는 한 개의 입력 변수로 하나의 결과를 예측하는 단순선형회귀를 먼저 배웁니다.

개념 2. 독립변수

독립변수는 예측에 사용하는 입력 변수입니다. 예를 들어 공부 시간으로 시험 점수를 예측한다면 공부 시간이 독립변수입니다.

개념 3. 종속변수

종속변수는 예측하려는 결과 변수입니다. 공부 시간과 점수 예제에서는 시험 점수가 종속변수입니다.

개념 4. 회귀직선

회귀직선은 데이터 점들의 경향을 가장 잘 나타내는 직선입니다. 단순선형회귀의 기본 식은 다음과 같습니다.

\[ \hat{y} = ax + b \]

여기서 \(a\)는 기울기, \(b\)는 절편, \(\hat{y}\)는 예측값입니다.

개념 5. 기울기

기울기는 독립변수 \(x\)가 1 증가할 때 예측값 \(\hat{y}\)가 얼마나 변하는지를 나타냅니다. 예를 들어 기울기가 6이면 공부 시간이 1시간 늘 때 예상 점수가 약 6점 증가한다고 해석할 수 있습니다.

개념 6. 절편

절편은 \(x=0\)일 때의 예측값입니다. 현실적으로 의미가 없을 수도 있으므로 무조건 해석하기보다 데이터 맥락을 고려해야 합니다.

개념 7. 잔차

잔차는 실제값과 예측값의 차이입니다. 잔차가 작을수록 모델이 데이터를 잘 설명한다고 볼 수 있습니다.

\[ \text{잔차} = y - \hat{y} \]

개념 8. 최소제곱법

최소제곱법은 잔차의 제곱합이 가장 작아지도록 회귀직선을 찾는 방법입니다. 잔차를 제곱하는 이유는 양수와 음수가 서로 상쇄되는 것을 막기 위해서입니다.

개념 9. 결정계수

결정계수 \(R^2\)는 회귀모델이 종속변수의 변동을 얼마나 설명하는지 나타내는 값입니다. 보통 0에서 1 사이의 값을 가지며, 1에 가까울수록 모델의 설명력이 높다고 해석합니다.

개념 10. 회귀분석의 한계

회귀분석은 데이터 범위 안에서의 경향을 바탕으로 예측합니다. 데이터 범위를 크게 벗어난 값을 예측하거나, 상관 관계를 인과 관계로 단정하면 잘못된 해석이 될 수 있습니다.

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 공부 시간과 점수 데이터 확인

study_hours = [1, 2, 3, 4, 5, 6]
scores = [55, 62, 70, 78, 86, 92]

for x, y in zip(study_hours, scores):
    print("공부 시간:", x, "점수:", y)

설명: 공부 시간은 독립변수, 시험 점수는 종속변수입니다.

코드 2. 산점도 그리기

import matplotlib.pyplot as plt

study_hours = [1, 2, 3, 4, 5, 6]
scores = [55, 62, 70, 78, 86, 92]

plt.scatter(study_hours, scores)
plt.title("공부 시간과 시험 점수")
plt.xlabel("공부 시간")
plt.ylabel("시험 점수")
plt.show()

설명: 산점도를 통해 두 변수 사이의 선형 경향을 눈으로 확인합니다.

코드 3. numpy로 회귀직선 구하기

import numpy as np

x = np.array([1, 2, 3, 4, 5, 6])
y = np.array([55, 62, 70, 78, 86, 92])

a, b = np.polyfit(x, y, 1)

print("기울기:", a)
print("절편:", b)
print("회귀식: y =", a, "* x +", b)

설명: np.polyfit(x, y, 1)은 데이터를 가장 잘 설명하는 1차식의 기울기와 절편을 구합니다.

코드 4. 회귀직선과 산점도 함께 그리기

import numpy as np
import matplotlib.pyplot as plt

x = np.array([1, 2, 3, 4, 5, 6])
y = np.array([55, 62, 70, 78, 86, 92])

a, b = np.polyfit(x, y, 1)
y_pred = a * x + b

plt.scatter(x, y, label="실제 데이터")
plt.plot(x, y_pred, label="회귀직선")
plt.title("단순선형회귀")
plt.xlabel("공부 시간")
plt.ylabel("시험 점수")
plt.legend()
plt.show()

설명: 산점도 위에 회귀직선을 그리면 모델이 데이터 경향을 얼마나 잘 따르는지 확인할 수 있습니다.

코드 5. 새로운 값 예측하기

import numpy as np

x = np.array([1, 2, 3, 4, 5, 6])
y = np.array([55, 62, 70, 78, 86, 92])

a, b = np.polyfit(x, y, 1)

new_x = 7
predicted_score = a * new_x + b

print("7시간 공부했을 때 예상 점수:", predicted_score)

설명: 회귀식을 이용하면 새로운 공부 시간에 대한 예상 점수를 계산할 수 있습니다.

코드 6. 잔차 계산하기

import numpy as np

x = np.array([1, 2, 3, 4, 5, 6])
y = np.array([55, 62, 70, 78, 86, 92])

a, b = np.polyfit(x, y, 1)
y_pred = a * x + b
residuals = y - y_pred

print("예측값:", y_pred)
print("잔차:", residuals)

설명: 잔차는 실제 점수와 예측 점수의 차이입니다.

코드 7. 결정계수 R² 계산하기

import numpy as np

x = np.array([1, 2, 3, 4, 5, 6])
y = np.array([55, 62, 70, 78, 86, 92])

a, b = np.polyfit(x, y, 1)
y_pred = a * x + b

ss_res = sum((y - y_pred) ** 2)
ss_tot = sum((y - y.mean()) ** 2)
r2 = 1 - ss_res / ss_tot

print("결정계수 R²:", r2)

설명: 결정계수는 회귀모델이 데이터 변동을 얼마나 설명하는지 보여 줍니다.

  • ⑤ 데이터가 사라진다.

R²가 높을수록 모델이 데이터 변동을 잘 설명한다고 볼 수 있습니다.

9. 회귀분석에서 주의해야 할 점은?

  • ① 데이터 범위를 크게 벗어난 예측은 위험할 수 있다.
  • ② 항상 모든 미래를 정확히 맞힌다.
  • ③ 잔차는 볼 필요가 없다.
  • ④ 산점도는 절대 보지 않는다.
  • ⑤ 이상치는 항상 좋은 데이터다.
정답: ①
관찰한 데이터 범위 밖의 예측은 불확실성이 커질 수 있습니다.

10. 회귀분석과 머신러닝의 관계로 옳은 것은?

  • ① 회귀분석은 예측 모델의 기초가 될 수 있다.
  • ② 회귀분석은 머신러닝과 전혀 관련이 없다.
  • ③ 회귀분석은 데이터 없이 가능하다.
  • ④ 회귀분석은 그래프를 사용할 수 없다.
  • ⑤ 회귀분석은 변수를 사용하지 않는다.
정답: ①
선형회귀는 머신러닝의 기본적인 지도학습 예측 모델 중 하나입니다.
Real Life Coding

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 공부 시간으로 시험 점수 예측

import numpy as np

study = np.array([1, 2, 3, 4, 5, 6])
score = np.array([55, 62, 70, 78, 86, 92])

a, b = np.polyfit(study, score, 1)
pred = a * 7 + b

print("7시간 공부 예상 점수:", round(pred, 2))

교육 데이터공부 시간과 점수의 관계를 이용해 새로운 공부 시간에 대한 예상 점수를 계산할 수 있습니다.

예제 2. 광고비로 매출 예측

import numpy as np

ad_cost = np.array([10, 20, 30, 40, 50])
sales = np.array([100, 150, 210, 260, 320])

a, b = np.polyfit(ad_cost, sales, 1)

print("광고비 60일 때 예상 매출:", round(a * 60 + b, 2))

마케팅 데이터광고비와 매출 사이에 선형 경향이 있다면 회귀분석으로 매출을 예측할 수 있습니다.

예제 3. 기온으로 전력 사용량 예측

import numpy as np

temperature = np.array([20, 22, 25, 28, 30, 32])
electricity = np.array([210, 220, 250, 300, 340, 380])

a, b = np.polyfit(temperature, electricity, 1)

print("기온 35도 예상 전력 사용량:", round(a * 35 + b, 2))

에너지 데이터기온이 높아질수록 전력 사용량이 늘어나는 경향을 회귀직선으로 표현할 수 있습니다.

예제 4. 회귀직선과 실제 데이터 비교

import numpy as np
import matplotlib.pyplot as plt

x = np.array([1, 2, 3, 4, 5])
y = np.array([3, 5, 7, 10, 11])

a, b = np.polyfit(x, y, 1)
y_pred = a * x + b

plt.scatter(x, y, label="실제값")
plt.plot(x, y_pred, label="회귀직선")
plt.legend()
plt.show()

시각화산점도와 회귀직선을 함께 보면 모델이 데이터의 경향을 잘 설명하는지 확인할 수 있습니다.

예제 5. 잔차로 예측 오차 확인

import numpy as np

x = np.array([1, 2, 3, 4, 5])
y = np.array([3, 5, 7, 10, 11])

a, b = np.polyfit(x, y, 1)
y_pred = a * x + b
residuals = y - y_pred

for actual, pred, res in zip(y, y_pred, residuals):
    print("실제:", actual, "예측:", round(pred, 2), "잔차:", round(res, 2))

모델 평가잔차를 확인하면 어떤 데이터에서 예측이 크게 빗나갔는지 살펴볼 수 있습니다.

Closing Talk

8. 수업을 마무리하는 멘트

오늘 우리는 회귀분석을 통해 데이터의 관계를 직선으로 표현하고 새로운 값을 예측하는 방법을 배웠습니다. 독립변수는 입력, 종속변수는 예측할 결과이며, 회귀직선의 기울기와 절편은 두 변수의 관계를 수학적으로 설명합니다. 컴퓨터과학은 파이썬으로 회귀모델을 빠르게 만들고 시각화하게 해 주며, 수학은 직선의 방정식과 최소제곱법의 원리를 제공합니다. 과학은 관찰 데이터로 가설을 세우고 예측을 검증하는 태도를 요구합니다. 데이터 과학과 머신러닝에서 회귀분석은 예측 모델의 가장 중요한 기초 중 하나입니다.

컴퓨터과학과의 연결

파이썬, numpy, matplotlib을 활용해 회귀직선을 계산하고 예측 결과를 시각화합니다.

수학과의 연결

일차함수, 기울기, 절편, 잔차, 제곱합은 회귀분석의 수학적 핵심입니다.

과학과의 연결

실험과 관찰 데이터를 바탕으로 변수 관계를 모델링하고 예측 결과를 검증합니다.

머신러닝과의 연결

선형회귀는 지도학습에서 연속적인 값을 예측하는 대표적인 머신러닝 모델입니다.
