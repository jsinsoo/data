2. 수업을 여는 멘트

여러분이 하루 동안 사용하는 앱들은 수많은 데이터를 자동으로 처리합니다. 음악 앱은 재생 기록을 저장하고, 쇼핑몰은 구매 기록을 분석하며, 날씨 앱은 기온 데이터를 계산해 알려 줍니다. 사람이 직접 모든 데이터를 하나씩 계산한다면 너무 오래 걸리겠지만, 파이썬을 사용하면 많은 데이터를 빠르고 정확하게 처리할 수 있습니다. 오늘은 데이터 분석의 도구가 되는 파이썬 기초 문법을 배웁니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 변수

변수는 데이터를 저장하는 이름표입니다. 예를 들어 score = 90에서 score는 변수이고 90은 값입니다. 데이터 분석에서는 온도, 점수, 판매량, 이름, 날짜 등을 변수에 저장하여 처리합니다.

개념 2. 자료형

자료형은 데이터의 종류입니다. 정수형 int, 실수형 float, 문자열 str, 참거짓 bool 등이 있습니다. 자료형을 이해해야 계산 가능한 데이터와 분류용 데이터를 구분할 수 있습니다.

개념 3. 리스트

리스트는 여러 데이터를 순서대로 저장하는 자료구조입니다. 시험 점수 목록, 일주일 기온, 상품별 판매량처럼 여러 값을 한 번에 다룰 때 유용합니다.

개념 4. 딕셔너리

딕셔너리는 키:값 형태로 데이터를 저장합니다. 학생 한 명의 이름, 점수, 희망 진로를 묶거나 상품명과 판매량을 연결할 때 적합합니다.

개념 5. 인덱싱과 슬라이싱

인덱싱은 리스트에서 특정 위치의 값을 꺼내는 방법이고, 슬라이싱은 여러 값을 범위로 잘라내는 방법입니다. 파이썬의 인덱스는 0부터 시작합니다.

개념 6. 조건문

조건문은 조건에 따라 다른 명령을 실행하는 구조입니다. 예를 들어 기온이 30도 이상이면 “더움”, 그렇지 않으면 “보통”으로 분류할 수 있습니다.

개념 7. 반복문

반복문은 여러 데이터를 하나씩 처리할 때 사용합니다. 점수 목록 전체를 출력하거나, 평균 계산을 위해 모든 값을 더하는 작업을 자동화할 수 있습니다.

개념 8. 함수

함수는 자주 사용하는 코드를 이름 붙여 재사용하는 방법입니다. 평균 계산 함수, 등급 판정 함수처럼 데이터 분석 과정을 정리할 수 있습니다.

개념 9. 데이터 분석 코드의 흐름

데이터 분석 코드는 일반적으로 데이터를 입력받고, 저장하고, 계산·분류하고, 결과를 출력하는 흐름을 가집니다. 이 흐름을 이해하면 복잡한 분석 프로그램도 단계적으로 만들 수 있습니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 변수와 값

temperature = 28
print("오늘 기온:", temperature, "도")

설명: temperature 변수에 28이라는 기온 데이터를 저장했습니다.

코드 2. 자료형 확인

score = 90
name = "민수"
passed = True

print(type(score))
print(type(name))
print(type(passed))

설명: type() 함수는 데이터가 정수, 문자열, 참거짓 중 어떤 자료형인지 알려 줍니다.

코드 3. 리스트로 점수 데이터 저장

scores = [80, 90, 75, 88, 92]

print("첫 번째 점수:", scores[0])
print("전체 점수:", scores)

설명: 리스트는 여러 점수를 하나의 이름으로 저장할 수 있습니다. 인덱스는 0부터 시작합니다.

코드 4. 딕셔너리로 학생 데이터 저장

student = {
    "이름": "지우",
    "수학": 91,
    "과학": 88,
    "희망진로": "데이터 과학자"
}

print(student["이름"])
print(student["희망진로"])

설명: 딕셔너리는 키를 사용하여 원하는 데이터를 쉽게 찾을 수 있습니다.

코드 5. 조건문으로 등급 판정

score = 85

if score >= 90:
    print("A등급")
elif score >= 80:
    print("B등급")
else:
    print("C등급 이하")

설명: 조건에 따라 점수를 등급으로 분류합니다.

코드 6. 반복문으로 데이터 합계 구하기

scores = [80, 90, 75, 88, 92]

total = 0
for s in scores:
    total += s

print("점수 합계:", total)

설명: 반복문은 리스트의 값을 하나씩 꺼내 자동으로 처리합니다.

코드 7. 함수로 평균 계산하기

def average(data):
    return sum(data) / len(data)

scores = [80, 90, 75, 88, 92]
print("평균:", average(scores))

설명: 평균 계산을 함수로 만들면 다른 데이터에도 반복해서 사용할 수 있습니다.

Real Life Coding

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 일주일 기온 평균 구하기

temperatures = [22, 24, 26, 25, 28, 30, 27]

average = sum(temperatures) / len(temperatures)

print("일주일 평균 기온:", round(average, 2))

날씨 데이터리스트에 저장된 기온 데이터를 평균으로 요약하여 한 주의 날씨 특징을 파악할 수 있습니다.

예제 2. 과목별 점수 딕셔너리 분석

scores = {
    "국어": 88,
    "수학": 92,
    "영어": 85,
    "과학": 95
}

best_subject = max(scores, key=scores.get)

print("가장 높은 과목:", best_subject)
print("점수:", scores[best_subject])

학습 데이터딕셔너리를 사용하면 과목명과 점수를 연결하여 가장 높은 과목을 쉽게 찾을 수 있습니다.

예제 3. 미세먼지 수치 등급 판정

dust = 87

if dust <= 30:
    level = "좋음"
elif dust <= 80:
    level = "보통"
elif dust <= 150:
    level = "나쁨"
else:
    level = "매우 나쁨"

print("미세먼지 등급:", level)

환경 데이터조건문을 사용하면 숫자 데이터를 생활 속 의사결정에 필요한 등급 정보로 바꿀 수 있습니다.

예제 4. 판매량이 높은 상품만 고르기

sales = {
    "물": 35,
    "주스": 28,
    "빵": 42,
    "우유": 19,
    "과자": 31
}

for item, count in sales.items():
    if count >= 30:
        print(item, ":", count, "개")

매점 데이터반복문과 조건문을 함께 사용하면 기준 이상의 상품만 자동으로 골라낼 수 있습니다.

예제 5. 평균 계산 함수 만들기

def average(data):
    return sum(data) / len(data)

steps = [5200, 6100, 7300, 6900, 8100]

print("평균 걸음 수:", average(steps))

건강 데이터평균 계산 함수를 만들면 점수, 기온, 걸음 수 등 다양한 데이터에 같은 분석 방법을 재사용할 수 있습니다.

8. 수업을 마무리하는 멘트
