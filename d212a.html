2. 수업을 여는 멘트

두 학급의 평균 점수가 모두 80점이라고 해 봅시다. A학급은 대부분 78~82점이고, B학급은 50점과 100점이 섞여 있습니다. 평균만 보면 두 학급은 같아 보이지만 실제 모습은 매우 다릅니다. 그래서 데이터 분석에서는 “중심”뿐 아니라 “얼마나 흩어져 있는가”도 반드시 살펴야 합니다. 오늘은 데이터의 안정성, 변동성, 예측 가능성을 판단하는 흩어짐의 측정 방법을 배웁니다.
Core Concepts

3. 각각의 개념에 대한 상세 설명

개념 1. 산포도

산포도는 데이터가 중심값 주변에 얼마나 퍼져 있는지를 나타내는 정도입니다. 평균이 같아도 산포도가 크면 값들이 넓게 흩어져 있고, 산포도가 작으면 값들이 평균 주변에 모여 있습니다.

개념 2. 범위

범위는 최댓값에서 최솟값을 뺀 값입니다. 계산이 매우 쉽지만, 최댓값과 최솟값 두 개만 사용하므로 이상치의 영향을 크게 받습니다.

\[ \text{범위} = \text{최댓값} - \text{최솟값} \]

개념 3. 편차

편차는 각 데이터 값이 평균에서 얼마나 떨어져 있는지를 나타냅니다. 어떤 값이 평균보다 크면 양의 편차, 평균보다 작으면 음의 편차를 가집니다.

\[ \text{편차} = x_i - \bar{x} \]

개념 4. 분산

분산은 편차를 제곱한 값들의 평균입니다. 편차를 그냥 더하면 양수와 음수가 상쇄되므로 제곱하여 흩어짐의 크기를 측정합니다.

\[ \text{분산} = \frac{\sum (x_i-\bar{x})^2}{n} \]

개념 5. 표준편차

표준편차는 분산의 제곱근입니다. 분산은 단위가 제곱되기 때문에 원래 데이터 단위로 해석하기 어렵습니다. 표준편차는 원래 단위와 같아 해석하기 더 쉽습니다.

\[ \text{표준편차} = \sqrt{\frac{\sum (x_i-\bar{x})^2}{n}} \]

개념 6. 사분위범위

사분위범위는 제3사분위수(Q3)에서 제1사분위수(Q1)를 뺀 값입니다. 데이터의 가운데 50%가 얼마나 퍼져 있는지 보여 주며, 이상치의 영향을 평균보다 덜 받습니다.

\[ \text{IQR} = Q_3 - Q_1 \]

개념 7. 이상치

이상치는 다른 값들과 크게 떨어져 있는 값입니다. 이상치는 입력 오류일 수도 있고, 특별한 사건일 수도 있습니다. 흩어짐을 해석할 때 이상치를 확인해야 합니다.

개념 8. 안정성과 변동성

표준편차가 작으면 데이터가 비교적 안정적이고 예측 가능성이 높다고 볼 수 있습니다. 표준편차가 크면 값의 변화가 크고 불확실성이 큽니다.

개념 9. 평균과 표준편차의 함께 해석

평균은 중심을 알려 주고, 표준편차는 흩어짐을 알려 줍니다. 데이터 분석에서는 평균과 표준편차를 함께 보아야 자료의 전체적인 특징을 더 정확하게 이해할 수 있습니다.

Python Codes

4. 각각의 개념과 상세 설명 관련 파이썬 코드

코드 1. 범위 구하기

data = [70, 75, 80, 85, 90]

data_range = max(data) - min(data)

print("최댓값:", max(data))
print("최솟값:", min(data))
print("범위:", data_range)

설명: 범위는 가장 큰 값과 가장 작은 값의 차이입니다.

코드 2. 편차 구하기

data = [70, 75, 80, 85, 90]

mean = sum(data) / len(data)
deviations = [x - mean for x in data]

print("평균:", mean)
print("편차:", deviations)

설명: 각 값이 평균에서 얼마나 떨어져 있는지 계산합니다.

코드 3. 분산 구하기

data = [70, 75, 80, 85, 90]

mean = sum(data) / len(data)
variance = sum((x - mean) ** 2 for x in data) / len(data)

print("분산:", variance)

설명: 편차를 제곱하여 평균을 내면 분산이 됩니다.

코드 4. 표준편차 구하기

data = [70, 75, 80, 85, 90]

mean = sum(data) / len(data)
variance = sum((x - mean) ** 2 for x in data) / len(data)
std = variance ** 0.5

print("표준편차:", std)

설명: 표준편차는 분산의 제곱근이며, 원래 데이터 단위로 해석하기 쉽습니다.

코드 5. 두 집단의 흩어짐 비교

class_a = [78, 79, 80, 81, 82]
class_b = [50, 65, 80, 95, 110]

def std(data):
    mean = sum(data) / len(data)
    variance = sum((x - mean) ** 2 for x in data) / len(data)
    return variance ** 0.5

print("A학급 평균:", sum(class_a) / len(class_a))
print("A학급 표준편차:", std(class_a))
print("B학급 평균:", sum(class_b) / len(class_b))
print("B학급 표준편차:", std(class_b))

설명: 두 학급의 평균이 같아도 표준편차가 다르면 데이터의 흩어짐이 다릅니다.

코드 6. 사분위범위 구하기

data = [50, 60, 70, 80, 90, 100, 110]

data.sort()
q1 = data[len(data)//4]
q3 = data[(len(data)*3)//4]
iqr = q3 - q1

print("Q1:", q1)
print("Q3:", q3)
print("IQR:", iqr)

설명: IQR은 데이터의 가운데 50%가 얼마나 퍼져 있는지 보여 줍니다.

코드 7. statistics 라이브러리 활용

import statistics as st

data = [70, 75, 80, 85, 90]

print("평균:", st.mean(data))
print("분산:", st.pvariance(data))
print("표준편차:", st.pstdev(data))

설명: 파이썬의 statistics 라이브러리를 사용하면 통계량을 간단히 구할 수 있습니다.

7. 실생활 관련 코드 예제 5개 + 상세설명

예제 1. 두 학급 점수 안정성 비교

class_a = [78, 79, 80, 81, 82]
class_b = [60, 70, 80, 90, 100]

def std(data):
    mean = sum(data) / len(data)
    return (sum((x - mean) ** 2 for x in data) / len(data)) ** 0.5

print("A학급 표준편차:", std(class_a))
print("B학급 표준편차:", std(class_b))

교육 데이터평균이 비슷해도 표준편차가 큰 학급은 학생 간 점수 차이가 크다고 해석할 수 있습니다.

예제 2. 일주일 기온 변동성 측정

temperatures = [21, 22, 25, 29, 30, 28, 24]

temp_range = max(temperatures) - min(temperatures)

print("기온 범위:", temp_range)

날씨 데이터기온 범위가 크면 일주일 동안 기온 변화가 컸다는 뜻입니다.

예제 3. 스마트폰 사용 시간 표준편차

hours = [2.5, 2.8, 3.0, 3.2, 7.0]

mean = sum(hours) / len(hours)
std = (sum((x - mean) ** 2 for x in hours) / len(hours)) ** 0.5

print("평균 사용 시간:", mean)
print("표준편차:", std)

생활 습관 데이터특정 날 사용 시간이 크게 늘어나면 표준편차가 커지고 생활 패턴이 불규칙하다고 해석할 수 있습니다.

예제 4. 상품 가격의 IQR

prices = [3000, 3200, 3300, 3500, 3700, 4000, 12000]

prices.sort()
q1 = prices[len(prices)//4]
q3 = prices[(len(prices)*3)//4]
iqr = q3 - q1

print("Q1:", q1)
print("Q3:", q3)
print("IQR:", iqr)

소비 데이터IQR은 가운데 가격대가 얼마나 퍼져 있는지 보여 주므로 극단적으로 비싼 상품의 영향을 줄여 해석할 수 있습니다.

예제 5. 운동 기록의 변동성 비교

week1 = [5200, 5400, 5300, 5500, 5600]
week2 = [3000, 8000, 4200, 9000, 2500]

def data_summary(data):
    mean = sum(data) / len(data)
    std = (sum((x - mean) ** 2 for x in data) / len(data)) ** 0.5
    return mean, std

print("1주차 평균과 표준편차:", data_summary(week1))
print("2주차 평균과 표준편차:", data_summary(week2))

건강 데이터걸음 수 평균뿐 아니라 표준편차를 보면 운동 습관이 꾸준한지 불규칙한지 파악할 수 있습니다.

8. 수업을 마무리하는 멘트

오늘 우리는 데이터의 중심을 나타내는 평균만으로는 충분하지 않다는 것을 배웠습니다. 범위는 가장 간단하게 흩어짐을 보여 주고, 편차는 각 값이 평균에서 얼마나 떨어졌는지 알려 주며, 분산과 표준편차는 전체 데이터의 변동성을 수치로 표현합니다. 컴퓨터과학은 파이썬을 통해 이러한 계산을 빠르고 정확하게 수행하게 해 주고, 수학은 흩어짐을 공식과 논리로 설명합니다. 과학에서는 실험 결과가 안정적인지 판단할 때 변동성 측정이 중요합니다. 데이터 과학과 머신러닝에서도 흩어짐을 이해해야 데이터의 신뢰성과 예측 가능성을 올바르게 판단할 수 있습니다.

컴퓨터과학과의 연결

파이썬 반복문, 함수, 통계 라이브러리를 활용해 분산과 표준편차를 자동 계산합니다.

수학과의 연결

편차, 제곱, 평균, 제곱근을 이용하여 데이터의 흩어짐을 정량적으로 표현합니다.

과학과의 연결

실험 측정값의 변동성을 확인하여 실험 결과의 안정성과 신뢰성을 판단합니다.

머신러닝과의 연결

데이터의 흩어짐과 이상치를 이해하면 모델 학습 전 데이터 정제와 특성 분석에 도움이 됩니다.
